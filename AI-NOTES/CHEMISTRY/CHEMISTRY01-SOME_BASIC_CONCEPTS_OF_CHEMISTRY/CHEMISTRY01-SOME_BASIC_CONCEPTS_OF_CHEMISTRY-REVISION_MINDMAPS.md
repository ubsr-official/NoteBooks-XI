## ⚖️ Laws of Chemical Combination — At a Glance

```
┌─────────────────────────────────────────────────────────────┐
│  1. Conservation of Mass (Lavoisier, 1789)                  │
│     Reactant mass = Product mass. Matter never created      │
│     or destroyed.                                           │
├─────────────────────────────────────────────────────────────┤
│  2. Definite Proportions (Proust)                           │
│     A compound ALWAYS has same element ratio by mass.       │
├─────────────────────────────────────────────────────────────┤
│  3. Multiple Proportions (Dalton, 1803)                     │
│     Two elements → multiple compounds → masses of one       │
│     element combine in simple whole number ratio.           │
│     Example: H₂O vs H₂O₂ → O ratio = 16:32 = 1:2          │
├─────────────────────────────────────────────────────────────┤
│  4. Gay Lussac's Law (1808)                                 │
│     Gases combine in simple volume ratios (same T & P).     │
│     H₂ : O₂ : H₂O = 2 : 1 : 2                             │
├─────────────────────────────────────────────────────────────┤
│  5. Avogadro's Law (1811)                                   │
│     Equal volumes of gases (same T & P) = equal molecules.  │
└─────────────────────────────────────────────────────────────┘
```

## 🔢 Significant Figures — Quick Rules

```
Non-zero digits          → ALWAYS significant  (285 = 3 SF)
Leading zeros            → NEVER significant   (0.0052 = 2 SF)
Zeros between digits     → ALWAYS significant  (2005 = 4 SF)
Trailing zeros + decimal → ALWAYS significant  (1.200 = 4 SF)
Trailing zeros, no dec.  → NOT significant     (1200 = 2 SF)
```

 **Calc rules** : Add/Subtract → match decimal places | Multiply/Divide → match fewest SF

## ⚛️ Mole Concept — All Formulas

$$
n = \frac{m}{M} = \frac{N}{N_A} = \frac{V(\text{gas at STP})}{22.7 \text{ L}}
$$

* **Nₐ = 6.022 × 10²³ mol⁻¹** (Avogadro's constant)
* 1 amu = 1.66056 × 10⁻²⁴ g
* Molar mass (g mol⁻¹) = numerically equal to atomic/molecular mass in u

---

## 🧫 % Composition & Formula

$$
\text{Mass % of element} = \frac{\text{Mass of element in 1 mol}}{\text{Molar mass of compound}} \times 100
$$

$$
n_{\text{(EF to MF)}} = \frac{\text{Molar Mass}}{\text{EF mass}} \quad \Rightarrow \quad \text{Molecular Formula} = n \times \text{EF}
$$

---

## 💧 Solution Concentrations

| Type               | Formula                            | Temp Dependent? |
| ------------------ | ---------------------------------- | --------------- |
| Molarity (M)       | n(solute) / V(L)                   | **YES**   |
| Molality (m)       | n(solute) / mass(kg of solvent)    | NO              |
| Mole fraction (χ) | nₐ / (nₐ + n_B)                  | NO              |
| Mass %             | mass(solute)/mass(solution) × 100 | NO              |

 **Dilution** : M₁V₁ = M₂V₂
