## ⚛️ MO Theory — Key Molecular Configurations

```
Molecule | Electrons | MO config (valence)                               | b.o. | Magnetic
─────────────────────────────────────────────────────────────────────────────────────────
H₂       | 2         | (σ1s)²                                             |  1   | Diamagnetic
He₂      | 4         | (σ1s)²(σ*1s)²                                      |  0   | DOES NOT EXIST
Li₂      | 6         | KK(σ2s)²                                           |  1   | Diamagnetic
B₂       | 10        | KK(σ2s)²(σ*2s)²(π2pₓ)¹(π2py)¹                   |  1   | Paramagnetic
C₂       | 12        | KK(σ2s)²(σ*2s)²(π2pₓ)²(π2py)²                   |  2   | Diamagnetic
N₂       | 14        | KK(σ2s)²(σ*2s)²(π2pₓ)²(π2py)²(σ2pz)²           |  3   | Diamagnetic
O₂       | 16        | KK(σ2s)²(σ*2s)²(σ2pz)²(π2pₓ)²(π2py)²           |  2   | Paramagnetic ⚠️
         |           | (π*2pₓ)¹(π*2py)¹                                  |      |
F₂       | 18        | KK(σ2s)²(σ*2s)²(σ2pz)²(π2pₓ)²(π2py)²           |  1   | Diamagnetic
         |           | (π*2pₓ)²(π*2py)²                                  |      |
Ne₂      | 20        | all filled                                         |  0   | DOES NOT EXIST
```

**MO energy order:**

* B₂, C₂, N₂: σ1s < σ*1s < σ2s < σ*2s < **π2pₓ = π2py** < σ2pz < π*2pₓ = π*2py < σ*2pz
* O₂, F₂: σ1s < σ*1s < σ2s < σ*2s < σ2pz < **π2pₓ = π2py** < π*2pₓ = π*2py < σ*2pz
  → KEY DIFFERENCE: σ2pz and π2p swap positions for lighter vs heavier molecules!

## ⚠️ HIGH-PRIORITY FACTS (Most Exam-Tested)

```
1. O₂ is PARAMAGNETIC (2 unpaired e⁻ in π*2p orbitals) — VB theory FAILS here!
   This is MO theory's greatest triumph.

2. F₂ bond is WEAKER than Cl₂ bond (F: 155 kJ mol⁻¹ vs Cl: 243 kJ mol⁻¹)
   Because F is so small that lone pair−lone pair repulsion is severe.

3. N₂ has the HIGHEST bond dissociation energy (946 kJ mol⁻¹) among diatomics.

4. C₂ has TWO PI BONDS and NO sigma bond between the carbons
   (Both highest-occupied MOs are π2p, σ2pz is empty in C₂)

5. NH₃ has HIGHER dipole moment than NF₃ despite F being more electronegative!
   Reason: Lone pair dipole in NH₃ adds to bond dipoles (same direction)
           Lone pair dipole in NF₃ opposes bond dipoles (opposite direction)

6. CO₂ has ZERO dipole moment despite polar C=O bonds (linear → cancel)
   But SO₂ is BENT → net dipole ≠ 0

7. Bond angle order: CH₄ (109.5°) > NH₃ (107°) > H₂O (104.5°)
   Each lone pair compresses the angle

8. PCl₅ axial bonds > equatorial bonds in LENGTH
   (more repulsion on axial → weaker → longer)

9. He₂, Be₂, Ne₂ DO NOT EXIST (bond order = 0)

10. ISOELECTRONIC species: N₂ ≡ CO ≡ NO⁺ ≡ CN⁻ (all 14e, bond order 3)
```

```
│ Conductivity                │ Good (molten/solution) vs Generally poor   │
│ Solubility                  │ In polar solvents vs In non-polar solvents │
│ State at RT                 │ Solid (crystalline) vs Gas/Liquid/Solid    │
│ Example                     │ NaCl, KCl, CaF₂ vs H₂, CH₄, CO₂, HCl    │
└─────────────────────────────┴────────────────────────────────────────────┘
```

## Lone Pair vs Bond Pair

```
┌─────────────────────────────┬──────────────────────────────────────────────┐
│ Feature                     │ Lone Pair vs Bond Pair                       │
├─────────────────────────────┼──────────────────────────────────────────────┤
│ Location                    │ On 1 atom (central) vs Between 2 atoms       │
│ Electron cloud size         │ Larger (concentrated) vs Smaller (spread out)│
│ Repulsion exerted           │ Greater vs Lesser                            │
│ Effect on geometry          │ Compresses bond angles vs Standard angles    │
│ Participation in bonding    │ No (non-bonding) vs Yes (bonding)            │
│ Lewis acid property         │ Donor (Lewis base) vs Not directly relevant  │
└─────────────────────────────┴──────────────────────────────────────────────┘
```

## Species with Same Bond Order (Isoelectronic Sets)

```
Bond order 1:  H₂, F₂, Cl₂, O₂²⁻ (peroxide), Li₂, Na₂
Bond order 2:  O₂, S₂, NO⁻, N₂²⁻
Bond order 2.5: O₂⁺, NO, NO⁻? (check carefully)
Bond order 3:  N₂, CO, NO⁺, CN⁻, BN (all 14 electrons, isoelectronic)

⚠️ Quick check for NO:
  NO has 15 electrons. b.o. = ½(10 − 5) = 2.5 (not 3!)
  NO⁺ has 14 electrons. b.o. = ½(10 − 4) = 3 (isoelectronic with N₂)
  NO⁻ has 16 electrons. b.o. = ½(10 − 6) = 2
```
