## 📊 Key Numbers and Data to Memorise

```
HYBRIDISATION QUICK-REFERENCE:
  sp³ → bond angle 109.5° → tetrahedral → example: CH₄, CH₃Cl, CH₃OH
  sp² → bond angle 120°   → trigonal planar → example: H₂C=CH₂, H₂C=O
  sp  → bond angle 180°   → linear → example: HC≡CH, HC≡N

s-CHARACTER: sp = 50% > sp² = 33.33% > sp³ = 25%
→ More s-character → shorter bond, stronger bond, more electronegative carbon

BOND TYPES:
  Single bond (C–C) = 1σ bond
  Double bond (C=C) = 1σ + 1π bond
  Triple bond (C≡C) = 1σ + 2π bonds

QUANTITATIVE ANALYSIS FORMULAS:
  % C = (12 × m_CO₂ × 100) / (44 × m)
  % H = (2 × m_H₂O × 100) / (18 × m)
  % N (Dumas) = (28 × V_STP × 100) / (22400 × m)   [V in mL, m in g]
  % N (Kjeldahl) = [1.4 × M × 2(V – V₁/2)] / m
  % X (Carius) = (at.mass X × m_AgX × 100) / (mol.mass AgX × m)
  % S = (32 × m_BaSO₄ × 100) / (233 × m)
  % P = (31 × m₁ × 100) / (1877 × m) [via phosphomolybdate]
  % O = 100 − (sum of all other percentages)

MOLAR MASSES FOR CARIUS:
  AgCl = 143.5 g/mol    AgBr = 188 g/mol    AgI = 235 g/mol

PRIORITY ORDER OF FUNCTIONAL GROUPS (highest → lowest):
  –COOH > –SO₃H > –COOR > –COCl > –CONH₂ > –CN > –CHO > >C=O > –OH > –NH₂ > C=C > C≡C

  Groups ALWAYS as prefix (never suffix):
  –R (alkyl), –C₆H₅ (phenyl), –X (halo), –NO₂ (nitro), –OR (alkoxy)
```

## 🔑 All Core Concepts at a Glance

```
┌────────────────────────────────────────────────────────────────────┐
│         BOND FISSION — HOMOLYTIC vs HETEROLYTIC                    │
│                                                                    │
│  HOMOLYTIC (Homolysis):                                            │
│    A:B → A• + •B  (one electron each)                              │
│    Products: FREE RADICALS (neutral, unpaired electron)            │
│    Arrow: half-headed (fish-hook) ⇀                               │
│    Reactions: FREE RADICAL / NONPOLAR reactions                    │
│    Stability: 3° > 2° > 1° > methyl radical                       │
│                                                                    │
│  HETEROLYTIC (Heterolysis):                                        │
│    A:B → A⁺ + :B⁻  or  A:B → :A⁻ + B⁺                           │
│    Products: CARBOCATIONS or CARBANIONS (ionic)                    │
│    Arrow: full-headed (double barb) →                              │
│    Reactions: IONIC / POLAR / HETEROPOLAR reactions                │
│    Carbocation: sp², trigonal planar, empty p-orbital              │
│    Carbanion: sp³, distorted tetrahedral, lone pair on C           │
│    Stability (carbocation): 3° > 2° > 1° > CH₃⁺                   │
├────────────────────────────────────────────────────────────────────┤
│            ELECTRON DISPLACEMENT EFFECTS                           │
│                                                                    │
│  INDUCTIVE EFFECT (I):                                             │
│    → Through σ-bonds; decreases with distance (negligible >3C)    │
│    –I groups (electron-withdrawing): –NO₂, –CN, –COOH, –X        │
│    +I groups (electron-donating): alkyl groups (3° > 2° > 1°)     │
│    Applications: acidity of RCOOH; carbocation stability          │
│                                                                    │
│  RESONANCE EFFECT (R/M):                                           │
│    → Through conjugated π-system; permanent                        │
│    +R: –NH₂, –OH, –OR, –X (halogens) — donate into π-system      │
│    –R: –NO₂, –CN, –CHO, >C=O — withdraw from π-system            │
│    Real molecule = resonance hybrid (lower energy than any         │
│                     single contributing structure)                 │
│                                                                    │
│  ELECTROMERIC EFFECT (E):                                          │
│    → TEMPORARY; only when attacking reagent is present             │
│    Complete transfer of π-electron pair to one atom                │
│    +E: π-electrons → towards atom that reagent attacks             │
│    –E: π-electrons → away from atom that reagent attacks           │
│    When I and E conflict: E-EFFECT PREDOMINATES                    │
│                                                                    │
│  HYPERCONJUGATION:                                                 │
│    → PERMANENT; σ(C–H) electrons delocalised into adjacent         │
│       π* or empty p-orbital                                        │
│    → "no-bond resonance"                                           │
│    More α C–H bonds → more hyperconjugation → more stable cation  │
│    CH₃⁺: 0 hyperconj. (p ⊥ to C–H) → LEAST stable               │
│    (CH₃)₃C⁺: 9 hyperconj. C–H bonds → MOST stable among simple   │
├────────────────────────────────────────────────────────────────────┤
│                  ISOMERISM SUMMARY                                 │
│                                                                    │
│  STRUCTURAL ISOMERISM (same MF, different structure):              │
│    Chain: different carbon skeletons                               │
│           e.g., pentane vs 2-methylbutane vs 2,2-dimethylpropane  │
│    Position: same skeleton, different substituent/FG position      │
│           e.g., propan-1-ol vs propan-2-ol                        │
│    Functional group: different FG entirely                         │
│           e.g., propanal vs propanone (C₃H₆O)                     │
│    Metamerism: same FG, different alkyl chains flanking it         │
│           e.g., CH₃OC₃H₇ vs C₂H₅OC₂H₅ (both C₄H₁₀O)            │
│                                                                    │
│  STEREOISOMERISM (same MF, same structure, different 3D):          │
│    Geometrical (cis-trans): due to restricted rotation             │
│    Optical: non-superimposable mirror images (chiral carbon)       │
├────────────────────────────────────────────────────────────────────┤
│               PURIFICATION TECHNIQUES                              │
│                                                                    │
│  Sublimation: sublimable + non-sublimable impurity                 │
│               examples: camphor, naphthalene, iodine               │
│  Crystallisation: solid with different solubility from impurity    │
│  Simple distillation: liquids with very different boiling points   │
│  Fractional distillation: close boiling points (petroleum)        │
│  Vacuum distillation: high BP liquid that decomposes               │
│                        example: glycerol                           │
│  Steam distillation: steam-volatile + water-immiscible             │
│                       example: aniline; basis: p₁+p₂ = P_atm     │
│  Differential extraction: separatory funnel; organic + aqueous    │
│  Chromatography: adsorption (TLC, column) or partition (paper)    │
│                   Rf = dist(substance)/dist(solvent)               │
├────────────────────────────────────────────────────────────────────┤
│             QUALITATIVE & QUANTITATIVE ANALYSIS                    │
│                                                                    │
│  C, H → CuO + heat → CO₂ (lime water) + H₂O (CuSO₄ turns blue)  │
│  N, S, X, P → Lassaigne's test (sodium fusion)                    │
│                                                                    │
│  LASSAIGNE'S:                                                      │
│    N → NaCN → Prussian blue with Fe₂(SO₄)₃/FeSO₄                 │
│    S → Na₂S → black PbS (Pb acetate) OR violet with nitroprusside │
│    X → NaX → AgX with AgNO₃ (Cl=white, Br=pale yellow, I=yellow) │
│    N+S together → NaSCN → blood-red with Fe³⁺ (no Prussian blue)  │
│    Test for X: add dil. HNO₃ FIRST (to decompose CN⁻, S²⁻)       │
│                                                                    │
│  QUANTITATIVE:                                                     │
│    Carbon+Hydrogen: combustion → CO₂ absorbed in KOH;            │
│                     H₂O absorbed in CaCl₂                         │
│    Nitrogen: Dumas (CuO, CO₂ atm, N₂ over KOH)                   │
│              Kjeldahl (H₂SO₄, → NH₃, absorbed in standard acid)   │
│    Halogens: Carius (fuming HNO₃, AgNO₃ → AgX)                   │
│    Sulphur: Carius/Na₂O₂ → BaSO₄ (BaCl₂)                         │
│    Phosphorus: HNO₃ → H₃PO₄ → phosphomolybdate (yellow)          │
│    Oxygen: by difference                                           │
└────────────────────────────────────────────────────────────────────┘
```

## 💡 Mnemonics and Memory Aids

```
FUNCTIONAL GROUP PRIORITY (for IUPAC naming):
  "Careless Students Often Compromise Class Honours, Avoiding Needless
   Knowledge Only Harms All Character"
  COOH, SO₃H, OCOR(ester), COCl, CONH₂, CN, CHO, C=O, OH, NH₂, C=C, C≡C

ELECTRON-WITHDRAWING (−I) GROUPS:
  "Nitro Cats Can Often Eat Arsenic" → –NO₂, –CN, –COOH, –OAr (aryloxy), –COOR, –F>Cl>Br>I

ELECTRON-DONATING (+I) GROUPS:
  "Alkyl Groups ADD Electrons To Carbon"
  All alkyl groups: tertiary > secondary > primary (more branching = stronger +I)

+R vs –R EFFECT (resonance):
  "+R gives electron RICHNESS to ring" → –NH₂, –OH, –OR, –halogens (lone pair donors)
  "–R REMOVES electrons from ring" → –NO₂, –CHO, –CN, –COOH, –C=O (π-acceptors)

LASSAIGNE'S TEST COLOURS:
  "P-B-V-Blood" → Prussian Blue (N), Black (S), Violet (S, nitroprusside), Blood-red (N+S)
  "White Yellow Yellow" → AgCl(white, Cl), AgBr(pale-yellow, Br), AgI(yellow, I)

CARBOCATION STABILITY:
  "The More Friends You Have, The More Stable You Are"
  More alkyl groups = more +I = more hyperconjugation = MORE STABLE
  3° > 2° > 1° > CH₃⁺ (NO hyperconjugation — zero friends)

HYBRIDISATION TRICK:
  Count bonds on carbon:
  4 single bonds only → sp³ (methane)
  1 double bond + 2 single → sp² (ethene)
  1 triple bond + 1 single → sp (ethyne)
  Also: a carbon bearing a lone pair and 2 bonds → sp² (carbene, carbocation)

HOMOLYTIC vs HETEROLYTIC:
  "HOMOlytic = HOMOgeneous split → each atom gets ONE electron"
  "HETEROlytic = HETEROGENEOUS split → one atom gets BOTH electrons"
  Free radicals = HOMO; Ions = HETERO

RF VALUE:
  "Rf = How FAR did the compound run relative to the solvent"
  Rf = spot distance / solvent distance
  0 < Rf < 1 always; high Rf = less adsorbed = more mobile

DUMAS vs KJELDAHL:
  "DUMAS = DRY method (gas collection)" — N₂ gas collected
  "KJELDAHL = WET method (liquid titration)" — NH₃ absorbed in acid
  KJELDAHL not for: Nitro, Azo, Ring-N (NAR = "Not All Ring-N")

STEAM DISTILLATION PRINCIPLE:
  "When TWO vapour pressures ADD UP to atmospheric, the mixture BOILS"
  p_organic + p_water = P_atm
  Since p_organic < P_atm alone → organic boils BELOW its own b.p.
```

## ⚠️ HIGH-PRIORITY DISTINCTIONS (Most Exam-Tested)

```
RESONANCE vs ISOMERISM:
  Resonance structures: SAME compound, different electron arrangements ONLY
                        nuclei in SAME positions; no physical reality alone
  Isomers: DIFFERENT compounds with different properties
  KEY TEST: If you can convert one structure to the other by ONLY moving
            electrons (no atoms move) → RESONANCE, not isomers

INDUCTIVE EFFECT vs RESONANCE EFFECT:
  Inductive: through σ-bonds; attenuates with distance; PERMANENT
  Resonance: through π-bonds (conjugated system); can act over long distances; PERMANENT
  Electromeric: through π-bonds; TEMPORARY (only when reagent is present)

NUCLEOPHILE vs ELECTROPHILE:
  Nucleophile = "nucleus lover" = ELECTRON DONOR (attacks δ+ centres)
               = has lone pairs or π-electrons
  Electrophile = "electron lover" = ELECTRON ACCEPTOR (attacks δ– centres)
               = electron-deficient (carbocations, BF₃, AlCl₃, >C=O)

HOMOLOGOUS SERIES vs ISOMERS:
  Homologues: DIFFERENT molecular formulas (differ by CH₂); same FG
  Isomers: SAME molecular formula; different structures/properties

CHAIN ISOMERISM vs POSITION ISOMERISM:
  Chain: the carbon SKELETON itself is different
  Position: same skeleton, just different POSITION of same FG/substituent

FRACTIONAL vs STEAM DISTILLATION:
  Fractional: for miscible liquids with close b.p. (e.g., alcohol + water)
  Steam: for immiscible liquid with water that is heat-sensitive (e.g., aniline)

COLUMN vs THIN LAYER CHROMATOGRAPHY:
  Column: preparative (can isolate larger amounts of compound)
  TLC: analytical (tests purity, identifies compounds by Rf)

DUMAS vs KJELDAHL METHOD:
  Dumas: more general; works for most N-containing compounds including nitro/azo/ring-N
  Kjeldahl: limited to amino/amide N; NOT for nitro, azo, or ring N (pyridine)
  Kjeldahl limitation: N must convert to (NH₄)₂SO₄ under H₂SO₄ conditions
```

## 📊 MIND MAP 3 — IUPAC Nomenclature Decision Tree

```
NAMING AN ORGANIC COMPOUND — DECISION FLOWCHART:

Step 1: IDENTIFY the principal functional group (highest in priority order)
        –COOH > –CHO > C=O > –OH > –NH₂ > C=C > C≡C
        (if no FG → parent is alkane)
        ↓
Step 2: FIND the longest chain containing the principal FG
        If cyclic → add "cyclo" prefix to alkane name
        ↓
Step 3: NUMBER the chain to give LOWEST locant to principal FG
        (if tie → give lowest locant to substituent nearest to one end)
        ↓
Step 4: NAME substituents as PREFIXES (alphabetical order)
        Identical substituents → di-, tri-, tetra- (NOT counted in alphabetical ordering)
        Branched substituents → name in parentheses, carbon attached to main chain = C1
        ↓
Step 5: WRITE the name: locant-prefix(es)-parent-locant-suffix
        e.g.: 6-methyl-octan-3-ol

SPECIAL RULES:
  • sec- and tert- NOT in alphabetical ordering; iso- and neo- ARE
  • For cyclic → lowest locant to most branched carbon
  • Two equal chains → choose chain with MORE substituents
  • Halogens, –NO₂, –OR always prefixes (never suffix)
  • Multiple same FG: add di/tri to suffix + keep terminal 'e' of alkane
    (e.g., hexane-2,4-dione; propane-1,2,3-triol)
  • Multiple C=C or C≡C: drop terminal 'e' of alkane before ene/yne
    (e.g., buta-1,3-diene)
```

WOHLER REACTION (1828):
  NH₄CNO ——Heat——→ NH₂CONH₂
  (Ammonium cyanate → Urea)

CARBON AND HYDROGEN DETECTION:
  C + 2CuO → 2Cu + CO₂  [CO₂ + Ca(OH)₂ → CaCO₃↓ (white turbidity)]
  2H + CuO → Cu + H₂O   [H₂O + anhydrous CuSO₄ (white) → CuSO₄·5H₂O (blue)]

LASSAIGNE'S TESTS:
  N: 6CN⁻ + Fe²⁺ → [Fe(CN)₆]⁴⁻; 3[Fe(CN)₆]⁴⁻ + 4Fe³⁺ → Fe₄[Fe(CN)₆]₃ (Prussian blue)
  S(a): S²⁻ + Pb²⁺ → PbS↓ (black)
  S(b): S²⁻ + [Fe(CN)₅NO]²⁻ → [Fe(CN)₅NOS]⁴⁻ (violet)
  N+S: Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺ (blood-red)
  X: X⁻ + Ag⁺ → AgX↓ (after acidifying with HNO₃)

STEAM DISTILLATION CONDITION:
  p_organic + p_water = P_atmospheric
  (Organic boils BELOW its normal boiling point)

Rf VALUE (Chromatography):
  Rf = distance moved by substance / distance moved by solvent (from same baseline)
  0 < Rf < 1 always
