## SECTION 1 — WHY DO WE NEED TO CLASSIFY ELEMENTS?

*"The Periodic Table is arguably the most important concept in chemistry."* — Glenn T. Seaborg

**The Problem of Abundance:**

* In 1800: only **31 elements** known
* By 1865: **63 elements** identified
* Currently: **114 elements** known (recently discovered ones are man-made)
* Efforts to synthesise new elements are ongoing

**Purpose of Classification:**

* Makes systematic study of all elements feasible
* Reveals recurring patterns and family relationships
* Explains known chemical facts rationally
* Predicts properties of undiscovered elements

> `[Board]` **Key Point:** Classification of elements is the foundation of chemistry — understanding one element in a group largely explains the whole family.

## SECTION 2 — GENESIS OF PERIODIC CLASSIFICATION

### 2.1 Dobereiner's Triads (1829) `[Board · NEET]`

**Scientist:** Johann Dobereiner (German chemist)

**Observation:** Groups of **three elements (triads)** share similar physical and chemical properties, and the **middle element's atomic weight is approximately the average** of the other two.

| Triad   | Element | Atomic Weight | Middle = Average?          |
| ------- | ------- | ------------- | -------------------------- |
| Triad 1 | Li      | 7             | Na ≈ (7+39)/2 = 23 ✓     |
|         | Na      | 23            |                            |
|         | K       | 39            |                            |
| Triad 2 | Ca      | 40            | Sr ≈ (40+137)/2 = 88.5 ✓ |
|         | Sr      | 88            |                            |
|         | Ba      | 137           |                            |
| Triad 3 | Cl      | 35.5          | Br ≈ (35.5+127)/2 = 81 ✓ |
|         | Br      | 80            |                            |
|         | I       | 127           |                            |

**Limitation:** Worked only for a **few** elements; dismissed as coincidence by many scientists.

> ⚠️ **Common Mistake:** Do not confuse Dobereiner's Triads with modern Group classification — Dobereiner's triads were based purely on atomic weight patterns, not electronic structure.

### 2.2 de Chancourtois' Cylindrical Table (1862) `[Board]`

**Scientist:** A.E.B. de Chancourtois (French geologist)

* Arranged known elements in order of **increasing atomic weights** on a **cylindrical table**
* Elements with similar properties fell on the same vertical line
* Showed **periodic recurrence of properties**
* Did not attract widespread attention at the time

### 2.3 Newlands' Law of Octaves (1865) `[Board · NEET]`

**Scientist:** John Alexander Newlands (English chemist)

**Law:** When elements are arranged in  **increasing order of atomic weights** , every **eighth element has properties similar to the first** — analogous to octaves in music.

**Newlands' Octaves:**

| Li | Be | B  | C  | N | O | F  |
| -- | -- | -- | -- | - | - | -- |
| Na | Mg | Al | Si | P | S | Cl |
| K  | Ca |    |    |   |   |    |

**Recognition:** Awarded the **Davy Medal in 1887** by the Royal Society, London (despite initial rejection).

**Limitation:** Law of Octaves worked only **up to calcium** — broke down for heavier elements.

### 2.4 Mendeleev's Periodic Table & Law (1869) `[Board · NEET · JEE]`

**Scientists:** Dmitri Mendeleev (Russian, 1834–1907) and Lothar Meyer (German, 1830–1895) — both proposed similar arrangements independently in 1869.

**Lothar Meyer's contribution:** Plotted **physical properties** (atomic volume, melting point, boiling point) against atomic weight → periodically repeated pattern.

**Mendeleev's Periodic Law:**

> *"The properties of the elements are a periodic function of their atomic weights."*

**Mendeleev's System:**

* Elements arranged in **horizontal rows** and **vertical columns**
* Elements with similar properties placed in the same **vertical column (group)**
* Used broader range of physical and chemical properties (including empirical formulas of compounds)

**Remarkable Predictions:** Mendeleev left **gaps** for undiscovered elements and predicted their properties:

| Property            | Eka-aluminium (predicted) | Gallium (found) | Eka-silicon (predicted) | Germanium (found) |
| ------------------- | ------------------------- | --------------- | ----------------------- | ----------------- |
| Atomic weight       | 68                        | 70              | 72                      | 72.6              |
| Density (g/cm³)    | 5.9                       | 5.94            | 5.5                     | 5.36              |
| Melting point       | Low                       | 302.93 K        | High                    | 1231 K            |
| Formula of oxide    | E₂O₃                    | Ga₂O₃         | EO₂                    | GeO₂             |
| Formula of chloride | ECl₃                     | GaCl₃          | ECl₄                   | GeCl₄            |

> 🔑 **Why Mendeleev is credited:** He published the Periodic Law  **first** , made **quantitative predictions** that were experimentally verified, and his boldness in leaving gaps (rather than forcing elements into wrong positions) was scientifically courageous.

**Anomalies in Mendeleev's Table:**

* Some elements placed in groups inconsistent with atomic weight order
  * e.g., Iodine (I, at. wt. 127) placed after Tellurium (Te, at. wt. 128) — reversed order justified by properties
* Position of isotopes unresolved
* Position of hydrogen (Group I or Group VII?) unclear

## SECTION 3 — MODERN PERIODIC LAW AND THE LONG FORM `[Board · NEET · JEE]`

### 3.1 Moseley's Contribution (1913)

**Scientist:** Henry Moseley (English physicist)

**Experiment:** Studied **characteristic X-ray spectra** of elements.

**Finding:** A plot of √ν (where ν = frequency of X-rays emitted) against **atomic number (Z)** gave a **straight line** — NOT the plot of √ν vs atomic mass.

**Conclusion:** Atomic number (Z) is a more **fundamental property** of an element than atomic mass.

### 3.2 Modern Periodic Law

> *"The physical and chemical properties of the elements are periodic functions of their  **atomic numbers** ."*

**Basis:** Periodicity is a consequence of the  **periodic variation in electronic configurations** .

### 3.3 Long Form of the Periodic Table

**Structure:**

* **7 horizontal rows** called **Periods** (period number = highest principal quantum number n)
* **18 vertical columns** called **Groups** or **Families** (IUPAC: numbered 1–18)

**Number of Elements per Period:**

| Period | n | Elements | Orbitals filled | Count    |
| ------ | - | -------- | --------------- | -------- |
| 1      | 1 | H to He  | 1s              | 2        |
| 2      | 2 | Li to Ne | 2s, 2p          | 8        |
| 3      | 3 | Na to Ar | 3s, 3p          | 8        |
| 4      | 4 | K to Kr  | 4s, 3d, 4p      | 18       |
| 5      | 5 | Rb to Xe | 5s, 4d, 5p      | 18       |
| 6      | 6 | Cs to Rn | 6s, 4f, 5d, 6p  | 32       |
| 7      | 7 | Fr to Og | 7s, 5f, 6d, 7p  | 32 (max) |

> 🔑 **Memory Formula:** Number of elements in a period =  **2 × (number of atomic orbitals in that energy level)** . For period 6: n=6 gives 6s (1), 4f (7), 5d (5), 6p (3) = 16 orbitals → 32 elements.

**14 elements each from 6th and 7th periods** (lanthanoids and actinoids respectively) are placed **separately at the bottom** to maintain the table's compact structure.

## SECTION 4 — IUPAC NOMENCLATURE FOR Z > 100 `[Board · NEET]`

**Why needed:** High-Z elements are extremely unstable, synthesised in tiny quantities, and naming disputes arise (e.g., Element 104: Americans named it  **Rutherfordium** , Soviets named it  **Kurchatovium** ).

**IUPAC Rule:** Derive name systematically from atomic number using  **digit roots** :

| Digit | Name | Abbreviation |
| ----- | ---- | ------------ |
| 0     | nil  | n            |
| 1     | un   | u            |
| 2     | bi   | b            |
| 3     | tri  | t            |
| 4     | quad | q            |
| 5     | pent | p            |
| 6     | hex  | h            |
| 7     | sept | s            |
| 8     | oct  | o            |
| 9     | enn  | e            |

**Rule:** Join roots in order of digits + **"ium"** suffix. Symbol = first three letters.

**Examples:**

| Z   | IUPAC Name   | Symbol | Official Name | Official Symbol |
| --- | ------------ | ------ | ------------- | --------------- |
| 101 | Unnilunium   | Unu    | Mendelevium   | Md              |
| 102 | Unnilbium    | Unb    | Nobelium      | No              |
| 104 | Unnilquadium | Unq    | Rutherfordium | Rf              |
| 106 | Unnilhexium  | Unh    | Seaborgium    | Sg              |
| 112 | Ununbium     | Uub    | Copernicium   | Cn              |
| 118 | Ununoctium   | Uuo    | Oganesson     | Og              |

**Problem 3.1 (NCERT):** Element with Z = 120?
→ Roots: 1=un, 2=bi, 0=nil → **Unbinilium (Ubn)**

> 🔑 **Exam Tip:** IUPAC systematic names use exactly 3-letter symbols. Official names may reflect country of discovery or honour scientists (e.g., Seaborgium honours Glenn T. Seaborg).

## SECTION 5 — ELECTRONIC CONFIGURATIONS AND THE PERIODIC TABLE `[Board · NEET · JEE]`

### 5.1 Electronic Configurations in Periods

**Rule:** The **period number** = the **principal quantum number (n)** of the outermost (valence) electrons.

**Period-by-Period Analysis:**

**Period 1 (n=1):** 1s fills → H (1s¹) to He (1s²) — 2 elements.

**Period 2 (n=2):** 2s then 2p fill → Li (2s¹) to Ne (2s²2p⁶) — 8 elements.

**Period 3 (n=3):** 3s then 3p fill → Na to Ar — 8 elements.

**Period 4 (n=4):** 4s fills first, then 3d (transition series), then 4p.

* 3d transition series: Sc (Z=21, 3d¹4s²) → Zn (Z=30, 3d¹⁰4s²)
* Period 4 ends at Kr —  **18 elements total** .

**Period 5 (n=5):** Similar to period 4. 4d transition series starts at Y (Z=39). Ends at Xe — 18 elements.

**Period 6 (n=6):** 6s, 4f, 5d, 6p fill in sequence.

* 4f-inner transition series (lanthanoid series): Ce (Z=58) → Lu (Z=71)
* Period 6 has **32 elements** (Cs to Rn).

**Period 7 (n=7):** 7s, 5f, 6d, 7p fill.

* 5f-inner transition series (actinoid series): Th (Z=90) → Lr (Z=103)
* Includes mostly man-made radioactive elements.
* Period 7 ends at **Og (Z=118)** — noble gas; 32 elements (theoretical maximum).

### 5.2 Groupwise Electronic Configurations

Elements in the **same group** have the **same valence shell electronic configuration** and thus similar chemical properties.

**Example — Group 1 (Alkali Metals):**

| Z  | Element | Electronic Configuration |
| -- | ------- | ------------------------ |
| 3  | Li      | [He]2s¹                 |
| 11 | Na      | [Ne]3s¹                 |
| 19 | K       | [Ar]4s¹                 |
| 37 | Rb      | [Kr]5s¹                 |
| 55 | Cs      | [Xe]6s¹                 |
| 87 | Fr      | [Rn]7s¹                 |

> **Key insight:** All alkali metals have **ns¹** valence configuration → one loosely held electron → high reactivity, form +1 ions.

## SECTION 6 — ELECTRONIC CONFIGURATIONS AND TYPES OF ELEMENTS: s, p, d, f BLOCKS `[Board · NEET · JEE]`

### 6.1 The s-Block Elements

**Groups:** 1 (alkali metals) and 2 (alkaline earth metals)
**Valence configuration:** ns¹ (Group 1) or ns² (Group 2)

**Key Characteristics:**

* All are **reactive metals** with **low ionization enthalpies**
* Lose outermost electrons readily: Group 1 → 1+ ion; Group 2 → 2+ ion
* **Metallic character and reactivity increase** going down the group
* Never found in pure form in nature (too reactive)
* Compounds (except Li and Be) are **predominantly ionic**
* Li and Be show **anomalous behaviour** due to small size and high charge-to-radius ratio

### 6.2 The p-Block Elements

**Groups:** 13 to 18
**Valence configuration:** ns²np¹ to ns²np⁶ (across each period)
**Also called:** Representative Elements or Main Group Elements (together with s-block)

**Key Characteristics:**

* Period ends with **noble gas** (closed valence shell: ns²np⁶) — very low chemical reactivity
* **Halogens** (Group 17) and **chalcogens** (Group 16): highly negative electron gain enthalpies; readily gain 1 or 2 electrons
* Non-metallic character **increases left → right** across a period
* Metallic character **increases down a group**

**Special Cases in p-block:**

* **Helium:** Strictly belongs to s-block (1s²) but placed in Group 18 (p-block) because its completely filled valence shell (1s²) gives it noble gas properties.
* **Hydrogen:** Only one s-electron like Group 1, but can also gain an electron like Group 17 (halogens). Placed separately at the top.

### 6.3 The d-Block Elements (Transition Elements)

**Groups:** 3 to 12
**General outer electronic configuration:** (n-1)d¹⁻¹⁰ ns⁰⁻² (exception: Pd = 4d¹⁰5s⁰)

**Key Characteristics:**

* All are **metals**
* Mostly form **coloured ions**
* Exhibit **variable valence** (oxidation states)
* Show **paramagnetism** (due to unpaired d electrons)
* Often used as **catalysts**
* Form a **bridge** between chemically active s-block metals and less active p-block elements (Groups 13–14)

**Important Exception:** Zn, Cd, Hg have configuration (n-1)d¹⁰ns² — completely filled d subshell → do NOT show most properties of transition elements.

**d-series:**

* 3d series: Sc (Z=21) to Zn (Z=30) — Period 4
* 4d series: Y (Z=39) to Cd (Z=48) — Period 5
* 5d series: La (Z=57)/Hf (Z=72) to Hg (Z=80) — Period 6
* 6d series: Ac (Z=89)/Rf (Z=104) onwards — Period 7

### 6.4 The f-Block Elements (Inner-Transition Elements)

**Two series:**

| Series      | Elements | Z range | Orbital filling | Period |
| ----------- | -------- | ------- | --------------- | ------ |
| Lanthanoids | Ce to Lu | 58–71  | 4f              | 6      |
| Actinoids   | Th to Lr | 90–103 | 5f              | 7      |

**General outer electronic configuration:** (n-2)f¹⁻¹⁴ (n-1)d⁰⁻¹ ns²

**Key Characteristics:**

* All are **metals**
* Properties within each series are **quite similar** (especially lanthanoids)
* Actinoid chemistry more complex (larger number of oxidation states possible)
* Actinoids are **radioactive**
* Many actinoids made only in **nanogram quantities** by nuclear reactions
* Elements after uranium are called **Transuranium Elements**

> 🔑 **Why placed separately at the bottom?** To maintain the compact structure of the table and preserve the principle of classifying elements with similar properties in the same column.

### 6.5 Metals, Non-metals, and Metalloids `[Board · NEET]`

**Metals (>78% of all elements):**

* Left side of Periodic Table
* Usually solids at room temperature (exceptions: Hg liquid; Ga, Cs melt just above room temperature)
* High melting and boiling points; good conductors of heat and electricity
* **Malleable** (beaten into sheets) and **ductile** (drawn into wires)

**Non-metals:**

* Upper right side of Periodic Table
* Solids or gases at room temperature (B and C are exceptions with high m.p.)
* Low melting and boiling points; poor conductors
* Most non-metallic solids are **brittle** — neither malleable nor ductile

**Metalloids (Semi-metals):**

* Elements bordering the **thick zig-zag line** in the table: Si, Ge, As, Sb, Te, Po (At debated)
* Properties intermediate between metals and non-metals
* Change from metallic to non-metallic character is gradual, not abrupt

**Trends in Metallic Character:**

* Increases going **down a group** (more metallic)
* Decreases going **left → right** across a period (less metallic)

> **Problem 3.4 (NCERT):** Increasing metallic character of P, Si, Be, Mg, Na:
> **Answer: P < Si < Be < Mg < Na**

## SECTION 7 — PERIODIC TRENDS IN PHYSICAL PROPERTIES `[Board · NEET · JEE]`

### 7.1 Atomic Radius `[Board · NEET · JEE]`

**Why difficult to measure precisely?**

1. Atom size (~1.2 Å) is extremely small
2. Electron cloud has **no sharp boundary** — probability never becomes exactly zero at finite distance

**Types of Atomic Radius:**

| Type                 | Definition                                                                        | Used for    |
| -------------------- | --------------------------------------------------------------------------------- | ----------- |
| Covalent Radius      | Half the internuclear distance between two identical atoms bonded by single bond  | Non-metals  |
| Metallic Radius      | Half the internuclear distance between adjacent metal atoms in metallic crystal   | Metals      |
| van der Waals Radius | Half the distance between nuclei of two adjacent non-bonded atoms of same element | Noble gases |

**Example:** Cl₂ bond distance = 198 pm → covalent radius of Cl = **99 pm**
**Example:** Cu metal: adjacent Cu-Cu = 256 pm → metallic radius = **128 pm**

> ⚠️ **Note:** Noble gas radii (van der Waals) are NOT comparable to covalent or metallic radii — they are much larger.

**Trends:**

| Direction          | Trend               | Reason                                                                                                                          |
| ------------------ | ------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Across period (→) | **Decreases** | Same valence shell but increasing nuclear charge (Zeff increases) → greater attraction of electrons to nucleus → smaller size |
| Down a group (↓)  | **Increases** | New principal quantum levels added → valence electrons further from nucleus; inner electrons shield nuclear charge             |

**Data — Atomic Radii Across Periods (in pm):**

| Period II | Li  | Be  | B  | C  | N  | O  | F  |
| --------- | --- | --- | -- | -- | -- | -- | -- |
| Radius    | 152 | 111 | 88 | 77 | 74 | 66 | 64 |

| Period III | Na  | Mg  | Al  | Si  | P   | S   | Cl |
| ---------- | --- | --- | --- | --- | --- | --- | -- |
| Radius     | 186 | 160 | 143 | 117 | 110 | 104 | 99 |

**Data — Atomic Radii Down Groups (in pm):**

| Group 1 | Li  | Na  | K   | Rb  | Cs  |
| ------- | --- | --- | --- | --- | --- |
| Radius  | 152 | 186 | 231 | 244 | 262 |

| Group 17 | F  | Cl | Br  | I   | At  |
| -------- | -- | -- | --- | --- | --- |
| Radius   | 64 | 99 | 114 | 133 | 140 |

### 7.2 Ionic Radius `[Board · NEET · JEE]`

**How measured:** From distances between cations and anions in ionic crystals.

**General Trend:** Same as atomic radii — increases down a group.

**Cations are SMALLER than parent atoms:**

* Fewer electrons but same nuclear charge → higher Zeff per electron → electrons pulled closer
* Example: Na = 186 pm → Na⁺ = **95 pm**

**Anions are LARGER than parent atoms:**

* More electrons → increased electron-electron repulsion → decreased effective nuclear charge → electrons spread out
* Example: F = 64 pm → F⁻ = **136 pm**

**Isoelectronic Species:** Atoms and ions with the  **same number of electrons** .

* Example: O²⁻, F⁻, Na⁺, Mg²⁺ all have **10 electrons**
* **Size order:** O²⁻ > F⁻ > Na⁺ > Mg²⁺ (greater nuclear charge → smaller size)

> 🔑 **Rule for isoelectronic species:** Greater the  **positive nuclear charge (Z)** , the **smaller** the ionic radius (more attraction of same electrons to nucleus).

**Problem 3.5 (NCERT):** Largest and smallest among Mg, Mg²⁺, Al, Al³⁺?

* Largest: **Mg** (neutral atom, largest radius)
* Smallest: **Al³⁺** (greatest positive charge in isoelectronic set)

### 7.3 Ionization Enthalpy (ΔᵢH) `[Board · NEET · JEE]`

**Definition:** Energy required to remove an electron from an  **isolated gaseous atom in its ground state** .

$$
X(g) \rightarrow X^+(g) + e^- \quad \Delta_i H_1 \text{ (first ionization enthalpy)}
$$

$$
X^+(g) \rightarrow X^{2+}(g) + e^- \quad \Delta_i H_2 \text{ (second ionization enthalpy)}
$$

**Units:** kJ mol⁻¹

**Key Facts:**

* Always **positive** (energy must be supplied)
* ΔᵢH₂ > ΔᵢH₁ > ΔᵢH₀ ... (successive IEs increase because removing from increasingly +ve ion)
* "Ionization enthalpy" without qualification = **first ionization enthalpy**

**Factors Governing Ionization Enthalpy:**

1. **Nuclear charge (Z):** Higher Z → stronger pull → higher IE
2. **Atomic radius:** Larger radius → electron further from nucleus → lower IE
3. **Shielding/Screening effect:** Inner electrons shield valence electron from nucleus → reduces effective nuclear charge (Zeff) → lower IE
4. **Orbital penetration:** s > p > d > f penetration to nucleus; s electrons held more tightly

**Trends:**

| Direction          | Trend                          | Reason                                                                                                              |
| ------------------ | ------------------------------ | ------------------------------------------------------------------------------------------------------------------- |
| Across period (→) | **Increases**(generally) | Nuclear charge increases; shielding by same-shell electrons minimal → Zeff increases → electron held more tightly |
| Down a group (↓)  | **Decreases**            | Increasing atomic radius; greater shielding by inner shells → Zeff felt by outermost electron decreases            |

**Anomalies Within Period 2:**

| Anomaly                                   | Explanation                                                                                                                                                                                 |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **B < Be**(IE of B is less than Be) | B removes a 2p electron; Be removes a 2s electron. 2p electron is more shielded from nucleus by 2s electrons (lower penetration) → easier to remove                                        |
| **O < N**(IE of O is less than N)   | N: three 2p electrons each in separate orbital (Hund's rule) — stable. O: one 2p orbital has paired electrons → electron-electron repulsion → easier to remove fourth 2p electron from O |

> ⚠️ **Actual IE order (Period 2):** Li < **B < Be** < C < **O < N** < F < Ne — two anomalies!

**Selected First Ionization Enthalpy Values (kJ mol⁻¹):**

| Period 2 | Li  | Be  | B   | C    | N    | O    | F    | Ne   |
| -------- | --- | --- | --- | ---- | ---- | ---- | ---- | ---- |
| ΔᵢH    | 520 | 899 | 801 | 1086 | 1402 | 1314 | 1681 | 2080 |

| Group 1 | Li  | Na  | K   | Rb  | Cs  |
| ------- | --- | --- | --- | --- | --- |
| ΔᵢH   | 520 | 496 | 419 | 403 | 374 |

**Noble gases:** Highest IE in each period (closed shell configuration — very stable)
**Alkali metals:** Lowest IE in each period (single ns¹ electron, shielded by noble gas core)

### 7.4 Electron Gain Enthalpy (ΔₑgH) `[Board · NEET · JEE]`

**Definition:** Enthalpy change when an electron is added to a **neutral gaseous atom** to form an anion.

$$
X(g) + e^- \rightarrow X^-(g) \quad \Delta_{eg}H
$$

* **Negative ΔₑgH:** Energy is **released** (exothermic) — stable anion forms (e.g., halogens)
* **Positive ΔₑgH:** Energy must be **supplied** (endothermic) — unstable anion (e.g., noble gases)

**Trends:**

| Direction          | Trend                                     | Explanation                                                                                    |
| ------------------ | ----------------------------------------- | ---------------------------------------------------------------------------------------------- |
| Across period (→) | Becomes**more negative**(generally) | Effective nuclear charge increases → added electron closer to nucleus → more energy released |
| Down a group (↓)  | Becomes**less negative**            | Atomic size increases → added electron farther from nucleus → less energy released           |

**Important Anomalies:**

* **F has LESS negative ΔₑgH than Cl** (F = −328 kJ mol⁻¹; Cl = −349 kJ mol⁻¹)
  * Reason: F is very small → added electron enters congested n=2 quantum level → significant electron-electron repulsion → less energy released.
  * For Cl (n=3), added electron occupies a larger region → less repulsion → more energy released.
* **O has LESS negative ΔₑgH than S** (similar reasoning as F vs Cl)

**Most negative ΔₑgH:** **Cl** (−349 kJ mol⁻¹) among all elements

**Electron Gain Enthalpy Data (kJ mol⁻¹):**

| Group 1 | ΔₑgH | Group 16 | ΔₑgH | Group 17 | ΔₑgH | Group 0 | ΔₑgH |
| ------- | ------ | -------- | ------ | -------- | ------ | ------- | ------ |
| H       | −73   |          |        |          |        | He      | +48    |
| Li      | −60   | O        | −141  | F        | −328  | Ne      | +116   |
| Na      | −53   | S        | −200  | Cl       | −349  | Ar      | +96    |
| K       | −48   | Se       | −195  | Br       | −325  | Kr      | +96    |
| Rb      | −47   | Te       | −190  | I        | −295  | Xe      | +77    |
| Cs      | −46   | Po       | −174  | At       | −270  | Rn      | +68    |

> **Problem 3.7 (NCERT):** Most negative ΔₑgH among P, S, Cl, F?
> **Answer: Cl** — because Cl has large 3p orbitals (less repulsion) and high Zeff; F has anomalously less negative ΔₑgH due to small size. **Least negative: P** (half-filled 3p³ — extra stability)

### 7.5 Electronegativity `[Board · NEET]`

**Definition:** A qualitative measure of the **ability of an atom in a chemical compound to attract shared electrons** towards itself.

**Key Distinction:** Not a measurable property of an isolated atom — it depends on the bonding context.

**Scales:** Pauling scale (most widely used), Mulliken-Jaffe scale, Allred-Rochow scale.

**Pauling Scale:**

* Fluorine assigned reference value of **4.0** (most electronegative element)
* Values range from ~0.7 (Cs) to 4.0 (F)

**Trends:**

| Direction          | Trend               | Reason                                                                                                     |
| ------------------ | ------------------- | ---------------------------------------------------------------------------------------------------------- |
| Across period (→) | **Increases** | Atomic radius decreases → valence electrons closer to nucleus → greater attraction for bonding electrons |
| Down a group (↓)  | **Decreases** | Atomic radius increases → valence electrons farther from nucleus → weaker attraction                     |

**Electronegativity Values (Pauling Scale):**

| Period II | Li  | Be  | B   | C   | N   | O   | F   |
| --------- | --- | --- | --- | --- | --- | --- | --- |
| χ        | 1.0 | 1.5 | 2.0 | 2.5 | 3.0 | 3.5 | 4.0 |

| Period III | Na  | Mg  | Al  | Si  | P   | S   | Cl  |
| ---------- | --- | --- | --- | --- | --- | --- | --- |
| χ         | 0.9 | 1.2 | 1.5 | 1.8 | 2.1 | 2.5 | 3.0 |

| Group 1 | Li  | Na  | K   | Rb  | Cs  |
| ------- | --- | --- | --- | --- | --- |
| χ      | 1.0 | 0.9 | 0.8 | 0.8 | 0.7 |

| Group 17 | F   | Cl  | Br  | I   | At  |
| -------- | --- | --- | --- | --- | --- |
| χ       | 4.0 | 3.0 | 2.8 | 2.5 | 2.2 |

**Relationship with Other Properties:**

* Electronegativity ∝ 1/atomic radius (inverse relationship)
* High electronegativity → non-metallic character
* Low electronegativity → metallic character
* Trend mirrors that of ionization enthalpy

## SECTION 8 — PERIODIC TRENDS IN CHEMICAL PROPERTIES `[Board · NEET · JEE]`

### 8.1 Periodicity of Valence (Oxidation States)

**Valence** = most characteristic property of an element; determined by electronic configuration.

**For Representative Elements:**

* Valence = number of electrons in outermost orbitals
* **OR** = 8 − (number of outermost electrons)

| Group             | 1 | 2 | 13 | 14 | 15  | 16  | 17  | 18  |
| ----------------- | - | - | -- | -- | --- | --- | --- | --- |
| Valence electrons | 1 | 2 | 3  | 4  | 5   | 6   | 7   | 8   |
| Valence           | 1 | 2 | 3  | 4  | 3,5 | 2,6 | 1,7 | 0,8 |

**Oxidation State vs Valence:**

* **Valence** = combining capacity (always positive, counting bonds)
* **Oxidation state** = charge assigned based on electronegativity considerations
* Example: Al in [AlCl(H₂O)₅]²⁺ → Oxidation state =  **+3** , Covalency = **6** (different!)

**Example Analysis:**

* OF₂: F is most electronegative → F = −1. Two F atoms, so O = **+2**
* Na₂O: O more electronegative than Na → O =  **−2** , Na = **+1**

### 8.2 Anomalous Properties of Second Period Elements `[Board · NEET · JEE]`

**Observation:** First element of groups 1, 2, and 13–17 **differs markedly** from other members of the same group.

**Reasons for Anomalous Behaviour:**

1. **Small atomic size** → high charge-to-radius ratio → unique bonding behaviour
2. **High electronegativity**
3. **Only four valence orbitals** (2s and 2p) → maximum covalency = 4
   * Later members have 9 valence orbitals (3s, 3p, 3d) → can expand octet
   * e.g., Boron forms [BF₄]⁻ (max 4 bonds) but Al forms [AlF₆]³⁻ (6 bonds)
4. First p-block elements form strong **pπ–pπ multiple bonds** (C=C, C≡C, N=N, N≡N, C=O, C=N, N=O)
   * Heavier elements (like Si, P) have poor overlap for multiple bonds → form single bonds + 3D structures

**Diagonal Relationship:**

* Li behaves like Mg; Be behaves like Al (more similar to *second element of the next group* than to members of their own group)
* Similarity arises because diagonal elements have similar charge-to-radius ratios

| Property              | Li  | Be  | B  | Mg  | Al  |
| --------------------- | --- | --- | -- | --- | --- |
| Metallic radius (pm)  | 152 | 111 | 88 | 160 | 143 |
| Ionic radius M⁺ (pm) | 76  | 31  | — | 72  | —  |

### 8.3 Periodic Trends and Chemical Reactivity `[Board · NEET]`

**Key Principle:** All chemical properties are manifestations of electronic configuration → periodicity in properties.

**Across a Period:**

* **IE lowest** at far left (alkali metals) → easily lose electrons → **most reactive metals**
* **ΔₑgH most negative** at far right (halogens) → easily gain electrons → **most reactive non-metals**
* Chemical reactivity:  **highest at both extremes** , lowest in the centre
* Alkali metals → form cations (oxidation), Halogens → form anions (reduction)

**Down a Group:**

* IE decreases → metallic character and reducing ability increase
* ΔₑgH less negative → non-metallic character and oxidising ability decrease

**Reactivity with Oxygen — Oxides:**

| Position in Period   | Oxide Type  | Example          | Behaviour with water           |
| -------------------- | ----------- | ---------------- | ------------------------------ |
| Far left (Group 1)   | Most basic  | Na₂O            | Na₂O + H₂O → 2NaOH          |
| Centre               | Amphoteric  | Al₂O₃, As₂O₃ | Reacts with both acid and base |
| Centre               | Neutral     | CO, NO, N₂O     | No acid/base reaction          |
| Far right (Group 17) | Most acidic | Cl₂O₇          | Cl₂O₇ + H₂O → 2HClO₄      |

> **Problem 3.10 (NCERT):** Na₂O is basic (→NaOH) and Cl₂O₇ is acidic (→HClO₄) — confirmed.

**Transition Metals (3d series):**

* Atomic radius change across period is **much smaller** than representative elements
* Ionization enthalpies are **intermediate** between s- and p-blocks
* Less electropositive than Groups 1 and 2

**Inner-Transition Metals:**

* Even smaller change in atomic radius across series (especially f-block)

## KEY DISTINCTIONS — HIGH-YIELD COMPARISONS `[NEET · JEE]`

| Mendeleev's Periodic Law                         | Modern Periodic Law                               |
| ------------------------------------------------ | ------------------------------------------------- |
| Properties are function of**atomic mass**  | Properties are function of**atomic number** |
| Based on observations and empirical data         | Based on electronic configuration theory          |
| Placed elements by atomic mass (with exceptions) | Places elements by atomic number (no exceptions)  |
| Could not explain position of isotopes           | Naturally explains isotopes (same Z, different A) |
| Fixed group VII for noble gases problem          | Noble gases fit naturally in Group 18             |

| Ionization Enthalpy                                 | Electron Gain Enthalpy                                    |
| --------------------------------------------------- | --------------------------------------------------------- |
| Energy to**remove**electron from neutral atom | Energy change when electron**added**to neutral atom |
| Always**positive**(endothermic)               | Can be negative (exothermic) or positive (endothermic)    |
| X(g) → X⁺(g) + e⁻                                | X(g) + e⁻ → X⁻(g)                                      |
| Measures tendency to**lose**electrons         | Measures tendency to**gain**electrons               |

| Electron Gain Enthalpy             | Electronegativity                             |
| ---------------------------------- | --------------------------------------------- |
| Property of**isolated atom** | Property of atom in a**compound**       |
| Measurable quantity (kJ mol⁻¹)   | Not directly measurable — comparative scale  |
| Refers to adding an electron       | Refers to attracting shared bonding electrons |
| Constant for a given atom          | Varies with bonding context                   |

| Cation                                       | Anion                                       |
| -------------------------------------------- | ------------------------------------------- |
| Formed by**loss**of electron(s)        | Formed by**gain**of electron(s)       |
| **Smaller**than parent atom            | **Larger**than parent atom            |
| Higher effective nuclear charge per electron | Lower effective nuclear charge per electron |
| Example: Na → Na⁺ (186 → 95 pm)           | Example: F → F⁻ (64 → 136 pm)            |
