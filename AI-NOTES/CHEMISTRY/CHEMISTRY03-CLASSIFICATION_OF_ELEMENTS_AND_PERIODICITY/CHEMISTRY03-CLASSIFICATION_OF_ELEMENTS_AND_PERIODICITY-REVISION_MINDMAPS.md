## 🔢 IUPAC Nomenclature for Z > 100 — Digit Roots

```
0=nil(n)  1=un(u)  2=bi(b)  3=tri(t)  4=quad(q)
5=pent(p) 6=hex(h) 7=sept(s) 8=oct(o)  9=enn(e)

Method: Chain digit roots + "ium"  →  Symbol = first 3 letters

Quick examples:
Z=101: un+nil+un+ium = Unnilunium (Unu) → Mendelevium (Md)
Z=104: un+nil+quad+ium = Unnilquadium (Unq) → Rutherfordium (Rf)
Z=118: un+un+oct+ium = Ununoctium (Uuo) → Oganesson (Og)
Z=120: un+bi+nil+ium = Unbinilium (Ubn)
```

## ⚠️ HIGH-PRIORITY ANOMALIES (Most Exam-Tested)

```
IONIZATION ENTHALPY ANOMALIES (Period 2):
  Actual order: Li < B < Be < C < O < N < F < Ne
  
  Anomaly 1: IE(B) < IE(Be)
  → B removes 2p electron; Be removes 2s electron
  → 2p is shielded by 2s (less penetrating) → easier to remove
  → Be(2s²) holds its electron more tightly than B(2p¹)

  Anomaly 2: IE(O) < IE(N)
  → N: 2p³ — three singly occupied orbitals (Hund) — extra stable
  → O: 2p⁴ — one paired orbital — electron-electron repulsion
  → Paired electron in O removed more easily despite higher Z

ELECTRON GAIN ENTHALPY ANOMALY:
  Cl (−349 kJ/mol) has MORE negative ΔegH than F (−328 kJ/mol)
  → F has very small n=2 shell → congested → added electron repelled
  → Cl has larger n=3 shell → less repulsion → more energy released
  → Most negative ΔegH overall: CHLORINE (Cl)

  Similarly: S has more negative ΔegH than O
```

```
─────────────────────────────────────────────────────────────────────────
  IONIC RADIUS RULES (extremely high exam frequency)
─────────────────────────────────────────────────────────────────────────

  CATION < PARENT ATOM:
  Na(186pm) → Na⁺(95pm)    Mg(160pm) → Mg²⁺(72pm)
  WHY: Fewer electrons, same Z → higher Zeff → pulled closer

  ANION > PARENT ATOM:
  F(64pm) → F⁻(136pm)    Cl(99pm) → Cl⁻(181pm)
  WHY: More electrons, same Z → lower Zeff → electrons spread out

  ISOELECTRONIC SERIES:  (all have same number of electrons)
  Example: O²⁻, F⁻, Na⁺, Mg²⁺ (all 10 electrons)
  Larger Z → SMALLER radius (more protons pull same electrons tighter)
  Order: O²⁻(Z=8) > F⁻(Z=9) > Na⁺(Z=11) > Mg²⁺(Z=12)
```
