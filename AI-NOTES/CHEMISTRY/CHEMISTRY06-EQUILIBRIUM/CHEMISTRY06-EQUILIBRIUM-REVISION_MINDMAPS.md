## 📊 Key Numbers to Memorise

```
R (gas constant for Kp) = 0.0831 bar L mol⁻¹ K⁻¹
Standard pressure = 1 bar (NOT 1 atm for Kp calculations)

Ionic product of water:
  Kw = [H⁺][OH⁻] = 1.0 × 10⁻¹⁴ M² at 298 K
  [H⁺] = [OH⁻] = 1.0 × 10⁻⁷ M in pure water
  pKw = 14 at 298 K; pH + pOH = 14

Strong acids (memorise all 6):
  HCl, HBr, HI, HNO₃, H₂SO₄, HClO₄

Strong bases:
  LiOH, NaOH, KOH, CsOH, Ba(OH)₂, Ca(OH)₂

Key Ka values (298 K):
  HF: 3.5 × 10⁻⁴       HNO₂: 4.5 × 10⁻⁴
  HCOOH: 1.8 × 10⁻⁴    CH₃COOH: 1.74 × 10⁻⁵ (pKa = 4.74)
  HCN: 4.9 × 10⁻¹⁰     Phenol: 1.3 × 10⁻¹⁰

Key Kb values (298 K):
  Dimethylamine: 5.4 × 10⁻⁴
  NH₃: 1.77 × 10⁻⁵ (pKb = 4.74)
  Aniline: 4.27 × 10⁻¹⁰

Ksp of BaSO₄ = 1.1 × 10⁻¹⁰ (sparingly soluble)
Ksp of AgCl = 1.8 × 10⁻¹⁰
```

## 🔑 All Core Equations at a Glance

```
┌────────────────────────────────────────────────────────────────────┐
│                  EQUILIBRIUM CONSTANTS                              │
│  Kc = [C]^c[D]^d / [A]^a[B]^b  (mol L⁻¹ units)                   │
│  Kp = pC^c · pD^d / pA^a · pB^b  (bar units)                      │
│  Kp = Kc(RT)^Δn                                                     │
│  Δn = mol gaseous products – mol gaseous reactants                  │
│  Reverse reaction: K'c = 1/Kc                                      │
│  Multiply equation by n: K_new = Kc^n                              │
│  Net reaction: KNET = K₁ × K₂ × ...                                │
├────────────────────────────────────────────────────────────────────┤
│                  IONIC EQUILIBRIA                                   │
│  Ka = [H⁺][A⁻]/[HA] = cα²/(1–α)  ≈ cα² (weak acid)              │
│  Kb = [M⁺][OH⁻]/[MOH] = cα²/(1–α)                                │
│  Ka × Kb = Kw = 10⁻¹⁴ at 298 K                                    │
│  pKa + pKb = pKw = 14 at 298 K                                    │
│  α = √(Ka/c)  [weak acid, α << 1]                                  │
├────────────────────────────────────────────────────────────────────┤
│                  pH CALCULATIONS                                    │
│  pH = –log[H⁺]        pOH = –log[OH⁻]                            │
│  pH + pOH = 14  (at 298 K)                                        │
│  Kw = [H⁺][OH⁻] = 10⁻¹⁴  (at 298 K)                             │
│  pKa = –log Ka         pKb = –log Kb                              │
├────────────────────────────────────────────────────────────────────┤
│                  BUFFER (Henderson-Hasselbalch)                     │
│  pH = pKa + log([A⁻]/[HA])  ← Acidic buffer                      │
│  pOH = pKb + log([BH⁺]/[B])  ← Basic buffer                      │
│  Best buffering: [A⁻] = [HA] → pH = pKa                           │
├────────────────────────────────────────────────────────────────────┤
│               SALT HYDROLYSIS                                       │
│  Strong acid + Strong base: pH = 7                                  │
│  Weak acid + Strong base: pH > 7                                    │
│  Strong acid + Weak base: pH < 7                                    │
│  Weak acid + Weak base: pH = 7 + ½(pKa – pKb)                    │
├────────────────────────────────────────────────────────────────────┤
│                  SOLUBILITY PRODUCT                                 │
│  MₓXᵧ → xM^p+ + yX^q–                                             │
│  Ksp = [M^p+]^x [X^q–]^y = (xS)^x(yS)^y = x^x·y^y·S^(x+y)     │
│  S = (Ksp / x^x·y^y)^[1/(x+y)]                                    │
│  IP > Ksp → precipitate; IP < Ksp → no precipitate                │
├────────────────────────────────────────────────────────────────────┤
│                  GIBBS – EQUILIBRIUM LINK                          │
│  ΔG° = –RT ln K = –2.303 RT log K                                 │
│  K = e^(–ΔG°/RT)                                                   │
└────────────────────────────────────────────────────────────────────┘
```

## 📐 Le Chatelier's Principle — Quick Summary

```
FACTOR          CHANGE          SHIFT          Kc AFFECTED?
─────────────────────────────────────────────────────────
Concentration   Add reactant    → Forward       No
                Remove reactant → Backward      No
                Add product     → Backward      No
                Remove product  → Forward       No

Pressure        Increase P      → Fewer mol gas No
(volume ↓)      Decrease P      → More mol gas  No
                Δn = 0          → No shift      No

Inert gas       At const. V     → No shift      No
addition        At const. P     → Toward more   No
                                  moles of gas

Temperature     Increase T      → Endothermic   YES ↑ or ↓
                                  direction     depending
                Decrease T      → Exothermic    YES
                                  direction

Catalyst        Added           → No shift      No
                                  (faster reach
                                   equilibrium)
─────────────────────────────────────────────────────────
KEY: Only temperature changes the VALUE of Kc!
```

## 🔢 Kc Magnitude and Extent of Reaction

```
VALUE OF Kc         EQUILIBRIUM POSITION
─────────────────────────────────────────
Kc > 10³           Products predominate
                   Reaction → nearly complete
                   Examples: H₂+Cl₂⇌2HCl (Kc=4×10³¹)

10⁻³ < Kc < 10³   Both reactants AND products
                   present in appreciable amounts
                   Examples: H₂+I₂⇌2HI at 700K (Kc=57)
                             N₂O₄⇌2NO₂ at 25°C (Kc=4.64×10⁻³)

Kc < 10⁻³          Reactants predominate
                   Reaction barely proceeds
                   Examples: N₂+O₂⇌2NO at 298K (Kc=4.8×10⁻³¹)
```

## ⚠️ HIGH-PRIORITY DISTINCTIONS (Most Exam-Tested)

```
Kc vs Kp:
  Kc: concentrations (mol L⁻¹); Kp: partial pressures (bar)
  Kp = Kc(RT)^Δn   R = 0.0831 bar L mol⁻¹ K⁻¹
  Kp = Kc ONLY when Δn = 0 (equal moles of gas on both sides)

Ka vs Kb:
  Ka: for weak acids; Kb: for weak bases
  Ka × Kb = Kw; pKa + pKb = 14 (at 298 K)
  Larger Ka → stronger acid → smaller pKa
  Larger Kb → stronger base → smaller pKb

STATIC vs DYNAMIC equilibrium:
  Static: no movement at all (NOT what happens in chemistry)
  Dynamic: forward and reverse BOTH ongoing at equal rates
  Evidence: Haber deuterium scrambling; radioactive sugar experiment

CATALYST effect:
  Speeds up BOTH forward and reverse reactions equally
  Reaches equilibrium FASTER but same Kc and composition
  Does NOT appear in Kc expression

ARRHENIUS vs BRØNSTED vs LEWIS:
  Arrhenius: limited to aqueous; H⁺/OH⁻ based
  Brønsted: proton transfer; works for non-aqueous too
  Lewis: broadest; electron pair transfer; includes non-proton acids (BF₃)

DEGREE OF IONIZATION (α):
  Increases with dilution (α = √Ka/c → as c↓, α↑)
  Decreases with common ion addition
  For strong acids: α = 1 (100%); for weak: << 1

Ksp COMPARISON between salts:
  CANNOT compare Ksp values of different formula salts directly
  Must calculate molar solubility S for each and compare S values
```

## 🔁 Kp–Kc Worked Decision Chart

```
Given reaction: N₂(g) + 3H₂(g) ⇌ 2NH₃(g)

Step 1: Count gaseous moles
  Reactants: 1 + 3 = 4 mol gas
  Products: 2 mol gas
  Δn = 2 – 4 = –2

Step 2: Apply formula
  Kp = Kc(RT)^Δn = Kc(RT)^(–2) = Kc/(RT)²

Step 3: Interpret
  Since Δn < 0: Kp < Kc (at any T > 0)
  At T = 500 K, RT = 0.0831×500 = 41.55
  So Kp = Kc / (41.55)² = Kc / 1726

USEFUL Δn EXAMPLES:
  H₂+I₂⇌2HI: Δn = 0 → Kp = Kc
  N₂O₄⇌2NO₂: Δn = +1 → Kp = Kc×RT > Kc
  N₂+3H₂⇌2NH₃: Δn = –2 → Kp = Kc/(RT)² < Kc
  CaCO₃(s)⇌CaO(s)+CO₂: Δn = 1 (only gas counts) → Kp = pCO₂
```

```

│                  PERIODIC TRENDS IN ACID STRENGTH            │
│                                                              │

│ H₂O << H₂S   (H₂S stronger acid than H₂O)                  │
│                                                              │
│ SAME PERIOD (going ACROSS): Electronegativity → stronger acid│
│ CH₄ < NH₃ < H₂O < HF  (acid strength increases →)          │
│ (Electronegativity of central atom increases →)              │
└──────────────────────────────────────────────────────────────┘
```

## 💡 Mnemonics and Memory Aids

```
THE SIX STRONG ACIDS:
  "HCl, HBr, HI, then the NICE ones: HNO₃, H₂SO₄, HClO₄"
  (Never Sour Pickles → Nitric, Sulfuric, Perchloric)

STRONG BASES:
  "Little Nasty Kids Can Be Annoying" 
  → LiOH, NaOH, KOH, CsOH, Ba(OH)₂, Ca(OH)₂

DIRECTION OF EQUILIBRIUM SHIFT (pressure):
  "Pressure pushes to FEWER moles of gas"
  Think: Pressure = squeeze → want less volume → fewer gas molecules

TEMPERATURE EFFECT:
  Think of heat as a REACTANT (endothermic) or PRODUCT (exothermic):
  Adding heat (↑T) → shifts AWAY from heat (Le Chatelier)
  Exothermic: heat is product → ↑T → backward (fewer products)
  Endothermic: heat is reactant → ↑T → forward (more products)

pH MEMORY:
  "Acidic is sour like A (pH < 7) → Neutral at 7 → Basic is bitter (pH > 7)"
  pH + pOH = 14 (think: 7+7=14, or 1+13=14)

CONJUGATE PAIR RULE:
  "Strong acid, WEAK conjugate base" (can't both be strong)
  Ka(acid) × Kb(conjugate base) = Kw = 10⁻¹⁴

BUFFER BEST POINT:
  "When salt equals acid, pH hits pKa"
  [A⁻] = [HA] → log(1) = 0 → pH = pKa

Ksp FORMULA PATTERN:
  "Write dissolution first, then multiply ion concentrations raised to coefficients"
  CaF₂ → Ca²⁺ + 2F⁻ → Ksp = [Ca²⁺][F⁻]² = S(2S)² = 4S³
```

## ⚡ Quick-Fire Formulas with Units

```
EQUILIBRIUM:
Kc = [products]^stoich / [reactants]^stoich  [dimensionless, but mol^Δn L^-Δn]
Kp = pproducts^stoich / preactants^stoich    [bar^Δn, dimensionless if bar/1bar]
Kp = Kc(RT)^Δn                              R = 0.0831 bar L mol⁻¹ K⁻¹

IONIC:
Ka = [H⁺][A⁻]/[HA]    [dimensionless (formally)]
Kb = [OH⁻][BH⁺]/[B]   [dimensionless]
Ka × Kb = Kw = 10⁻¹⁴   [at 298 K]
pKa + pKb = 14          [at 298 K]
α = √(Ka/c)             [dimensionless, 0 < α < 1]

pH CALCULATIONS:
pH = –log[H⁺]           [dimensionless]
pOH = –log[OH⁻]         [dimensionless]
pH + pOH = 14            [at 298 K]
For strong acid HX: pH = –log[HX]
For weak acid: [H⁺] = √(Ka × c)    → pH = ½(pKa – log c)

BUFFER:
pH = pKa + log([A⁻]/[HA])   [Henderson–Hasselbalch]

SOLUBILITY:
S(AgCl) = √Ksp               [mol L⁻¹]
S(CaF₂) = (Ksp/4)^(1/3)    [mol L⁻¹]
S(Fe(OH)₃) = (Ksp/27)^(1/4) [mol L⁻¹]
```
