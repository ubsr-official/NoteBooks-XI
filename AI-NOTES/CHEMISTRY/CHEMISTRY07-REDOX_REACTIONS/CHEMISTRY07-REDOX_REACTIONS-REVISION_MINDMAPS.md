## 📊 Key Numbers to Memorise

```
STANDARD ELECTRODE POTENTIALS (E° in Volts at 298 K):
  SHE (reference):          H⁺/H₂              =  0.00 V
  Strongest oxidant:        F₂/F⁻              = +2.87 V
  Common oxidants:          MnO₄⁻/Mn²⁺         = +1.51 V (acidic)
                            Cl₂/Cl⁻             = +1.36 V
                            Cr₂O₇²⁻/Cr³⁺       = +1.33 V
                            Br₂/Br⁻             = +1.09 V
                            Ag⁺/Ag              = +0.80 V
                            Fe³⁺/Fe²⁺           = +0.77 V
                            I₂/I⁻               = +0.54 V
                            Cu²⁺/Cu             = +0.34 V
  Common reductants:        Zn²⁺/Zn             = –0.76 V
                            Al³⁺/Al             = –1.66 V
                            Na⁺/Na              = –2.71 V
  Strongest reductant:      Li⁺/Li              = –3.05 V

KEY MEMORY TRICK:
  Above H₂ in series (positive E°) = stronger oxidant = weaker reductant
  Below H₂ in series (negative E°) = weaker oxidant = stronger reductant
```

## 🔑 All Core Equations at a Glance

```
┌────────────────────────────────────────────────────────────────────┐
│              DEFINITIONS — THREE VIEWS                             │
│                                                                    │
│  CLASSICAL VIEW:                                                   │
│    Oxidation = addition of O / electroneg. element                 │
│              = removal of H / electropositive element              │
│    Reduction = removal of O / electroneg. element                  │
│              = addition of H / electropositive element             │
│                                                                    │
│  ELECTRONIC VIEW:                                                  │
│    Oxidation = Loss of electrons        → OIL                     │
│    Reduction = Gain of electrons        → RIG                     │
│    Oxidising agent = electron acceptor (gets reduced)              │
│    Reducing agent = electron donor (gets oxidised)                 │
│                                                                    │
│  OXIDATION NUMBER VIEW:                                            │
│    Oxidation = INCREASE in O.N.                                    │
│    Reduction = DECREASE in O.N.                                    │
├────────────────────────────────────────────────────────────────────┤
│            OXIDATION NUMBER RULES (SUMMARY)                        │
│                                                                    │
│  Free element → O.N. = 0                                           │
│  Monoatomic ion → O.N. = charge                                    │
│  O: –2 (usual); –1 (peroxide, H₂O₂, Na₂O₂)                       │
│     –½ (superoxide, KO₂, RbO₂)                                    │
│     +2 (OF₂); +1 (O₂F₂)                                           │
│  H: +1 (usual); –1 (metal hydrides: LiH, NaH, CaH₂)              │
│  F: ALWAYS –1                                                      │
│  Alkali metals in compounds: ALWAYS +1                             │
│  Alkaline earth metals: ALWAYS +2                                  │
│  Al in compounds: ALWAYS +3                                        │
│  Sum in neutral compound = 0                                       │
│  Sum in ion = charge on ion                                        │
├────────────────────────────────────────────────────────────────────┤
│                  TYPES OF REDOX REACTIONS                          │
│                                                                    │
│  1. COMBINATION: A(element) + B → C                               │
│     e.g., 2Mg + O₂ → 2MgO; C + O₂ → CO₂                         │
│                                                                    │
│  2. DECOMPOSITION: C → A(element) + B                             │
│     e.g., 2H₂O → 2H₂ + O₂; 2KClO₃ → 2KCl + 3O₂                 │
│     [NOTE: CaCO₃ → CaO + CO₂ is NOT redox]                        │
│                                                                    │
│  3. DISPLACEMENT: X + YZ → XZ + Y                                 │
│     Metal: Zn + CuSO₄ → ZnSO₄ + Cu                               │
│     Non-metal (H): Zn + 2HCl → ZnCl₂ + H₂                       │
│     Halogen: Cl₂ + 2KBr → 2KCl + Br₂                             │
│                                                                    │
│  4. DISPROPORTIONATION: Same element oxidised AND reduced          │
│     e.g., 2H₂O₂ → 2H₂O + O₂ (O: –1 → –2 and 0)                  │
│            Cl₂ + 2OH⁻ → ClO⁻ + Cl⁻ + H₂O (Cl: 0 → +1 and –1)   │
├────────────────────────────────────────────────────────────────────┤
│              BALANCING METHODS                                     │
│                                                                    │
│  OXIDATION NUMBER METHOD:                                          │
│    Increase in O.N. = Decrease in O.N. (electrons conserved)      │
│    Adjust coefficients to equalise changes                         │
│    Then balance H and O with H⁺/OH⁻ and H₂O                       │
│                                                                    │
│  HALF-REACTION METHOD:                                             │
│    Split → Balance atoms → Balance O (H₂O) → Balance H (H⁺)       │
│    → Balance charge (e⁻) → Equalise e⁻ → Add halves               │
│    Acidic: use H⁺ directly                                         │
│    Basic: use H⁺ then add OH⁻ to both sides → H₂O                 │
├────────────────────────────────────────────────────────────────────┤
│                  ELECTRODE PROCESSES                               │
│                                                                    │
│  Redox couple: Oxidised form / Reduced form   e.g., Zn²⁺/Zn       │
│  Anode: OXIDATION (negative in galvanic cell)                      │
│  Cathode: REDUCTION (positive in galvanic cell)                    │
│  E° (SHE) = 0.00 V by convention                                   │
│  E° > 0 → weaker reductant than H₂ (stronger oxidant)             │
│  E° < 0 → stronger reductant than H₂                              │
│  Feasibility: Higher E° species oxidises lower E° species          │
└────────────────────────────────────────────────────────────────────┘
```

## ⚡ Halogen Activity Order (EXAM FAVOURITE)

```
OXIDISING POWER:    F₂ >> Cl₂ > Br₂ > I₂   (decreases down group 17)
REDUCING POWER:     HF << HCl < HBr < HI    (increases down group 17 for HX)
                    (Hydroiodic acid = BEST REDUCTANT among HX)

DISPLACEMENT RULES:
  F₂ displaces Cl⁻, Br⁻, I⁻ (but reacts with water — done electrolytically)
  Cl₂ displaces Br⁻ and I⁻ (Layer Test: Br₂ — orange; I₂ — violet in CCl₄)
  Br₂ displaces I⁻ only
  I₂ cannot displace any halide

CANNOT DISPLACE BY CHEMICAL MEANS:
  F⁻ → F₂: Only by ELECTROLYSIS (F₂ is strongest oxidant; no chemical can oxidise F⁻)
```

## ⚠️ HIGH-PRIORITY DISTINCTIONS (Most Exam-Tested)

```
OXIDATION vs REDUCTION:
  Often confused — remember OIL RIG
  Oxidising agent: gets REDUCED (gains e⁻; O.N. DECREASES)
  Reducing agent: gets OXIDISED (loses e⁻; O.N. INCREASES)
  These are commonly swapped in MCQs!

DISPROPORTIONATION vs NORMAL REDOX:
  Normal redox: two DIFFERENT elements change O.N.
  Disproportionation: SAME element in SAME compound is both oxidised and reduced
  Test: Does one species appear on both sides of the arrow as the element
  that changes? → YES = disproportionation

COMBINATION vs DECOMPOSITION:
  Combination: must have ELEMENTAL reactant(s) for it to be redox
  Decomposition: must have ELEMENTAL product(s) for it to be redox
  Counter-example: CaO + CO₂ → CaCO₃ (combination, NOT redox — no O.N. changes)
  Counter-example: CaCO₃ → CaO + CO₂ (decomposition, NOT redox)

ELECTRODE POTENTIAL:
  More POSITIVE E° = STRONGER OXIDISING AGENT (top of electrochemical series)
  More NEGATIVE E° = STRONGER REDUCING AGENT (bottom of electrochemical series)
  Feasibility: oxidant must have HIGHER E° than reductant
  
GALVANIC vs ELECTROLYTIC CELL:
  Galvanic: spontaneous reaction → electrical energy (ΔG < 0)
  Electrolytic: non-spontaneous, electrical energy drives reaction (ΔG > 0)
  In BOTH: Cathode = reduction; Anode = oxidation (definition)
  BUT: Cathode is +ve in galvanic; cathode is –ve in electrolytic

FRACTIONAL O.N.:
  AVERAGE value; electrons never actually share in fractions
  Always look for the structure to know the real whole-number O.N.
  Fe₃O₄ → Fe: +8/3 average (really Fe²⁺ + 2Fe³⁺)
  O₂⁺ and O₂⁻: genuinely O = +½ and –½ (these ARE fractional, not averages)
```

## 🔢 Electrochemical Series — Predicting Feasibility

```
RULE: The species HIGHER in the table (more positive E°) is the oxidant
      The species LOWER in the table (more negative E°) is the reductant
      Reaction: Oxidant (top) + Reductant (bottom) → Products [FEASIBLE]

FEASIBILITY EXAMPLES:
  Fe³⁺(+0.77) + I⁻(0.54) → Fe²⁺ + I₂    ✓ Fe³⁺ above I₂ in table
  Ag⁺(+0.80) + Cu(+0.34) → Ag + Cu²⁺    ✓ Ag⁺ above Cu²⁺ in table
  Fe³⁺(+0.77) + Cu(+0.34) → Fe²⁺ + Cu²⁺ ✓ Fe³⁺ above Cu in table
  Ag(+0.80) + Fe³⁺(+0.77) → Ag⁺ + Fe²⁺  ✗ Ag⁺ higher than Fe³⁺; Ag NOT reductant
  Br₂(+1.09) + Fe²⁺(+0.77) → Br⁻ + Fe³⁺ ✓ Br₂ above Fe³⁺

INCREASING REDUCING POWER (bottom of table wins):
  Ag < Hg < Cu < Fe²⁺ < H₂ < Pb < Ni < Fe < Cr < Zn < Al < Mg < Na < Li
```

```
│ Combination    │ A(elem.) + B → C        │ ✓ YES if elemental   │
│                │ CaO + CO₂ → CaCO₃       │ ✗ NO (no elem.)      │
├────────────────┼─────────────────────────┼──────────────────────┤
│ Decomposition  │ C → A + B(elem.)        │ ✓ YES if elem. formed│
│                │ CaCO₃ → CaO + CO₂       │ ✗ NO (no elem.)      │
├────────────────┼─────────────────────────┼──────────────────────┤
│ Displacement   │ More active replaces     │ ✓ ALWAYS redox       │
│                │ less active element      │                      │
├────────────────┼─────────────────────────┼──────────────────────┤
│ Disproportionat│ Same element: ↑ and ↓  │ ✓ ALWAYS redox       │
│ ion            │ O.N. simultaneously      │                      │
├────────────────┼─────────────────────────┼──────────────────────┤
│ Double displace│ AgNO₃ + NaCl → AgCl↓   │ ✗ NO — only ion      │
│ (non-redox)    │ + NaNO₃                  │ exchange             │
└────────────────┴─────────────────────────┴──────────────────────┘
```

## Which Metals React with Water/Steam/Acids?

```
┌─────────────────────┬──────────────────────────────────────────┐
│ Metal Group          │ Reactivity                               │
├─────────────────────┼──────────────────────────────────────────┤
```

```
OXIDATION vs REDUCTION:
  OIL RIG: Oxidation Is Loss (of e⁻); Reduction Is Gain (of e⁻)
  AN OX, RED CAT: ANode = OXidation; REDuction at CAThode

STOCK NOTATION MEMORY:
  "Put the ROMAN NUMERAL for the METAL's OXIDATION STATE in BRACKETS"
  Fe(II)Cl₂ — iron is +2; Fe(III)Cl₃ — iron is +3

HALOGEN OXIDISING POWER:
  "FIBIC" → Fluorine Is Best In Class (strongest oxidant)
  Or simply: Go DOWN group 17 → oxidising power DECREASES

DISPROPORTIONATION — SPOTTING IT:
  "The element SPLITS — goes UP and DOWN simultaneously"
  Look for: intermediate O.N. element → products at both higher AND lower O.N.
  CANNOT happen: F₂ (can't go positive) and ClO₄⁻ (Cl already at max +7)

ELECTROCHEMICAL SERIES — STRONG REDUCTANTS:
  "Lithium Never Misses Calcium As A Friend In Zinc, Leading Heroes Constantly"
  Li, Na, Mg, Ca, Al, Fe, Zn, Pb, H, Cu, Ag → increasing E° (decreasing reducing power)

OXYGEN O.N. — THE EXCEPTION CHAIN:
  "Usually –2, but in PEroxides it's –1, in SUpEroxides –½,
   and with Fluorine it's POSITIVE (+2 in OF₂)"
  Memory: PE SU FP → Peroxide –1; Superoxide –½; with Fluorine = Positive

H IN HYDRIDES — REMEMBER:
  "Hydrogen is POSITIVE (+1) everywhere EXCEPT in METAL HYDRIDES where it's NEGATIVE (–1)"
  LiH, NaH, CaH₂: H = –1

SALT BRIDGE FUNCTION:
  "The BRIDGE keeps CHARGE BALANCE: cations to cathode, anions to anode"
  Without it → charge build-up → current stops
```

## ⚡ Quick-Fire Formulas with Notes

```
OXIDATION NUMBER CALCULATIONS:
  Find x in compound: use algebraic sum = 0 (compound) or = charge (ion)
  H₂SO₄: 2(+1) + x + 4(–2) = 0 → x = +6 (S)
  Cr₂O₇²⁻: 2x + 7(–2) = –2 → x = +6 (Cr)
  MnO₄⁻: x + 4(–2) = –1 → x = +7 (Mn)
  Fe₃O₄: 3x + 4(–2) = 0 → x = +8/3 (Fe) [average]

BALANCING KEY CHECKS (before finalising):
  ✓ Total electrons in oxidation half = total in reduction half
  ✓ Atoms of each element balance on both sides
  ✓ Total charge balance on both sides

DANIELL CELL:
  E°cell = E°(cathode) – E°(anode) = E°(Cu²⁺/Cu) – E°(Zn²⁺/Zn)
         = +0.34 – (–0.76) = +1.10 V
  E°cell > 0 → spontaneous reaction

FEASIBILITY RULE:
  Reaction: Oxidant + Reductant → Products
  If E°(oxidant) > E°(reductant in table) → FEASIBLE
  If E°(oxidant) < E°(reductant in table) → NOT FEASIBLE
```
