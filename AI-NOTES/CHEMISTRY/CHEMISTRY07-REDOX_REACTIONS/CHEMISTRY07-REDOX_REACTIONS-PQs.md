**Q81.** **[Numerical]** Calculate the oxidation number of sulphur in H₂SO₅ (Caro's acid / peroxomonosulphuric acid) and explain why it appears anomalous.

**Solution:**
H₂SO₅ contains a peroxo linkage (–O–O–). Structure: H–O–O–S(=O)₂–OH

Let x = O.N. of S:
2(+1) + S + 5(–2) = 0 gives S = +8 — IMPOSSIBLE (max +6 for S)

**Reason for fallacy:** One oxygen is part of peroxo bond (O.N. = –1 each for those 2 O atoms), not –2.
Correct: 2(+1) + S + 3(–2) + 2(–1) = 0 → 2 + S – 6 – 2 = 0 → **S = +6** ✓

The structure shows the correct accounting: two peroxo oxygens (–O–O–) at –1 each, three regular oxygens at –2. The apparent +8 is a fallacy when we assign all O as –2.

**Q82.** **[Numerical]** In Ostwald's process: 4NH₃(g) + 5O₂(g) → 4NO(g) + 6H₂O(g)

Starting with 10.00 g NH₃ and 20.00 g O₂, calculate:
(a) Which is the limiting reagent?
(b) Maximum mass of NO that can be obtained?

**Solution:**

→ 0.588 mol NH₃ needs 0.588 × (5/4) = **0.735 mol O₂**

**Q83.** **[Numerical]** Balance the following equation in basic medium by the half-reaction method:

P₄(s) + OH⁻(aq) → PH₃(g) + H₂PO₂⁻(aq)

**Solution:**

**Oxidation state check:**
* P in P₄ = 0
* P in PH₃ = –3 (reduced)
* P in H₂PO₂⁻: 2(+1) + P + 2(–2) = –1 → P = +1 (oxidised)

**Oxidation half:** P → H₂PO₂⁻
P⁰ → P^(+1) + e⁻
Balance: P + 2H₂O → H₂PO₂⁻ + 2H⁺ + e⁻ ... then convert for base:
P + 2H₂O + e⁻ (? no, this is oxidation)
After full treatment: P – e⁻ → P^+1

**Reduction half:** P → PH₃
P + 3e⁻ → P^(–3); balance with H⁺:
P + 3H₂O + 3e⁻ → PH₃ + 3OH⁻ ... (in basic medium)

**Equalising electrons** (LCM = 3):
Reduction (×1): P + 3H₂O + 3e⁻ → PH₃ + 3OH⁻
Oxidation (×3): 3P + 6H₂O → 3H₂PO₂⁻ + 6H⁺ + 3e⁻ → in base: 3P + 6H₂O + 6OH⁻ → 3H₂PO₂⁻ + 6H₂O (simplify) → 3P → 3H₂PO₂⁻ + 6H⁺ – 6OH⁻... 

**Final balanced equation:**
**P₄ + 3OH⁻ + 3H₂O → PH₃ + 3H₂PO₂⁻**

*(Note: 1 P is reduced to PH₃; 3 P are oxidised to H₂PO₂⁻ — disproportionation of P₄)*

**Q84.** **[Numerical]** Identify the type of redox reaction and find the change in oxidation number for each species in:

2NO₂(g) + 2OH⁻(aq) → NO₂⁻(aq) + NO₃⁻(aq) + H₂O(l)

**Solution:**

N in NO₂: 2(–2) + N = 0 → N = **+4**
N in NO₂⁻: 2(–2) + N = –1 → N = **+3** (reduced, change = –1)
N in NO₃⁻: 3(–2) + N = –1 → N = **+5** (oxidised, change = +1)

The same element N goes from +4 simultaneously to +3 (reduction) and +5 (oxidation) → This is a **disproportionation reaction**.

Electrons: 1 e⁻ gained (N: +4→+3) and 1 e⁻ lost (N: +4→+5) per molecule pair → balanced.

**Q85.** **[Numerical]** Predict whether the following reactions are feasible using the standard electrode potential table:

(a) Fe³⁺(aq) + I⁻(aq) → Fe²⁺(aq) + ½I₂(s) [E°(Fe³⁺/Fe²⁺) = +0.77 V; E°(I₂/I⁻) = +0.54 V]
(b) Ag(s) + Fe³⁺(aq) → Ag⁺(aq) + Fe²⁺(aq) [E°(Ag⁺/Ag) = +0.80 V]
(c) Br₂(aq) + Fe²⁺(aq) → 2Br⁻(aq) + Fe³⁺(aq) [E°(Br₂/Br⁻) = +1.09 V]

**Solution:**

A redox reaction is feasible when the species with the **higher** E° acts as the oxidising agent (oxidises the species with lower E°).

**(a)** E°(Fe³⁺/Fe²⁺) = +0.77 V > E°(I₂/I⁻) = +0.54 V → Fe³⁺ can oxidise I⁻ → **FEASIBLE** ✓

**(b)** E°(Fe³⁺/Fe²⁺) = +0.77 V < E°(Ag⁺/Ag) = +0.80 V → Fe³⁺ cannot oxidise Ag (Ag would need to lose electrons to Ag⁺, but Fe³⁺ is weaker oxidant than Ag⁺) → **NOT FEASIBLE** ✗

**(c)** E°(Br₂/Br⁻) = +1.09 V > E°(Fe³⁺/Fe²⁺) = +0.77 V → Br₂ can oxidise Fe²⁺ to Fe³⁺ → **FEASIBLE** ✓
