## SECTION 1 — CLASSICAL IDEA OF REDOX REACTIONS `[Board · NEET]`

*"Where there is oxidation, there is always reduction — Chemistry is essentially a study of redox systems."*

### 1.1 Oxidation — The Classical View

**Originally defined** as the **addition of oxygen** to an element or compound:

| Reaction | Process |
|----------|---------|
| 2Mg(s) + O₂(g) → 2MgO(s) | Mg is oxidised (O₂ added) |
| S(s) + O₂(g) → SO₂(g) | S is oxidised (O₂ added) |
| CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l) | C–H oxidised (O₂ added, H₂ removed) |

**Definition broadened** to include:
* **Removal of hydrogen** from a substance: 2H₂S(g) + O₂(g) → 2S(s) + 2H₂O(l) — H₂S is oxidised
* **Addition of an electronegative element**: Mg + F₂ → MgF₂; Mg + Cl₂ → MgCl₂
* **Removal of an electropositive element**: 2K₄[Fe(CN)₆] + H₂O₂ → 2K₃[Fe(CN)₆] + 2KOH (K removed from ferrocyanide)

> **Final Classical Definition of Oxidation:** Addition of oxygen/electronegative element to a substance **OR** removal of hydrogen/electropositive element from a substance.

### 1.2 Reduction — The Classical View

**Originally:** removal of oxygen from a compound.

**Broadened to include:**

| Reaction | Reduction process |
|----------|------------------|
| 2HgO(s) → 2Hg(l) + O₂(g) | Removal of oxygen from HgO |
| 2FeCl₃(aq) + H₂(g) → 2FeCl₂(aq) + 2HCl(aq) | Removal of electronegative Cl from FeCl₃ |
| CH₂=CH₂(g) + H₂(g) → H₃C–CH₃(g) | Addition of hydrogen |
| 2HgCl₂(aq) + SnCl₂(aq) → Hg₂Cl₂(s) + SnCl₄(aq) | Addition of Hg (electropositive) to HgCl₂ |

> **Final Classical Definition of Reduction:** Removal of oxygen/electronegative element from a substance **OR** addition of hydrogen/electropositive element to a substance.

### 1.3 Key Insight — Simultaneous Nature

> "It was soon realised that oxidation and reduction **always occur simultaneously**, hence the word **REDOX** was coined for this class of chemical reactions."

In reaction 2HgCl₂ + SnCl₂ → Hg₂Cl₂ + SnCl₄:
* HgCl₂ is reduced (Hg receives a partner)
* SnCl₂ is simultaneously oxidised to SnCl₄ (Cl, an electronegative element, is added)

## SECTION 2 — REDOX REACTIONS IN TERMS OF ELECTRON TRANSFER `[Board · NEET · JEE]`

### 2.1 The Electronic Interpretation

When sodium reacts with chlorine, oxygen, or sulphur:

```
2Na(s) + Cl₂(g) → 2NaCl(s) → Na⁺Cl⁻(s)
4Na(s) + O₂(g) → 2Na₂O(s) → (Na⁺)₂O²⁻(s)
2Na(s) + S(s) → Na₂S(s)  → (Na⁺)₂S²⁻(s)
```

These ionic compounds show **charge development** → implies electron transfer.

**Splitting into half-reactions for NaCl formation:**

```
Oxidation half:  2Na(s) → 2Na⁺(g) + 2e⁻
Reduction half:  Cl₂(g) + 2e⁻ → 2Cl⁻(g)
─────────────────────────────────────────
Overall:         2Na(s) + Cl₂(g) → 2Na⁺Cl⁻(s)
```

Each separate step is called a **half-reaction**, which explicitly shows the electrons.

### 2.2 Electronic Definitions

> **Oxidation:** Loss of electron(s) by any species. → **OIL** (Oxidation Is Loss)
>
> **Reduction:** Gain of electron(s) by any species. → **RIG** (Reduction Is Gain)
>
> **Oxidising agent (Oxidant):** Acceptor of electron(s). [It gets reduced itself]
>
> **Reducing agent (Reductant):** Donor of electron(s). [It gets oxidised itself]

**Memory mnemonic:**

```
OIL RIG
Oxidation Is Loss | Reduction Is Gain
```

**In reactions 2Na + Cl₂ → 2NaCl:**
* Na loses electrons → Na is **oxidised** → Na is the **reducing agent**
* Cl₂ gains electrons → Cl₂ is **reduced** → Cl₂ is the **oxidising agent**

### 2.3 Competitive Electron Transfer Reactions

**Zinc in copper sulphate solution:**

```
Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s)
```

* Strip becomes coated with reddish copper
* Blue colour of CuSO₄ disappears
* Zn loses electrons → oxidised; Cu²⁺ gains electrons → reduced
* Equilibrium **greatly favours products**

**Copper in silver nitrate:**

```
Cu(s) + 2Ag⁺(aq) → Cu²⁺(aq) + 2Ag(s)
```

* Solution turns blue (Cu²⁺ formed)
* Silver deposits on copper rod
* Equilibrium greatly favours products: Cu²⁺(aq) and Ag(s)

**Cobalt in nickel sulphate:**

```
Co(s) + Ni²⁺(aq) → Co²⁺(aq) + Ni(s)
```

* At equilibrium, both Ni²⁺(aq) and Co²⁺(aq) are present at **moderate concentrations**
* Neither reactants nor products are greatly favoured

**Trend observed:**

```
Electron releasing tendency: Zn > Cu > Ag
```

This forms the basis of the **electrochemical series (metal activity series)**.

> **Key Idea:** Competition for electrons is analogous to competition for protons among acids. A table of metals listed by their tendency to release electrons = **electrochemical series**.

## SECTION 3 — OXIDATION NUMBER `[Board · NEET · JEE]`

### 3.1 What is Oxidation Number?

**Oxidation number** denotes the oxidation state of an element in a compound, ascertained according to a set of rules, formulated on the basis that the **electron pair in a covalent bond belongs entirely to the more electronegative element**.

> This is a **book-keeping device** — it assumes complete electron transfer even in covalent bonds. In reality, it is only an electron shift.

For example:

```
  0    0   +1 –2
2H₂ + O₂ → 2H₂O
```

H goes from 0 → +1 (oxidised); O goes from 0 → –2 (reduced).

### 3.2 Rules for Assigning Oxidation Number

**Rule 1:** In free/uncombined elemental state, oxidation number = **zero**.
> H₂, O₂, Cl₂, O₃, P₄, S₈, Na, Mg, Al → all are **0**.

**Rule 2:** For monoatomic ions, oxidation number = **charge on the ion**.
> Na⁺ → +1; Mg²⁺ → +2; Fe³⁺ → +3; Cl⁻ → –1; O²⁻ → –2

**Rule 3:** Oxidation number of **oxygen = –2** in most compounds.
Exceptions:
* Peroxides (H₂O₂, Na₂O₂): O = **–1**
* Superoxides (KO₂, RbO₂): O = **–½**
* When bonded to fluorine (OF₂): O = **+2**; (O₂F₂): O = **+1**

**Rule 4:** Oxidation number of **hydrogen = +1**, except in **metal hydrides** (binary compounds with metals):
> LiH, NaH, CaH₂ → H = **–1**

**Rule 5:** **Fluorine** always = **–1** (most electronegative element, no positive state possible). Other halogens (Cl, Br, I) = –1 in halides, but can have **positive values** in oxoacids and oxoanions (when combined with O).

**Rule 6:** Sum of oxidation numbers of all atoms in a **compound = 0**. In a **polyatomic ion** = charge on the ion.
> Example: CO₃²⁻ → C + 3(–2) = –2 → C = +4

### 3.3 Summary Table — Common Oxidation Numbers

| Element | Usual O.N. | Exception / Range |
|---------|------------|-------------------|
| O | –2 | Peroxide: –1; Superoxide: –½; OF₂: +2 |
| H | +1 | Metal hydrides: –1 |
| F | –1 | Always –1 (no exception) |
| Cl, Br, I | –1 (halide) | Positive in oxoacids/oxoanions |
| Na, K | +1 | Always +1 in compounds |
| Mg, Ca | +2 | Always +2 in compounds |
| Al | +3 | Always +3 in compounds |

### 3.4 Highest Oxidation States (Period 3)

| Group | 1 | 2 | 13 | 14 | 15 | 16 | 17 |
|-------|---|---|----|----|----|----|-----|
| Element | Na | Mg | Al | Si | P | S | Cl |
| Compound | NaCl | MgSO₄ | AlF₃ | SiCl₄ | P₄O₁₀ | SF₆ | HClO₄ |
| Highest O.N. | +1 | +2 | +3 | +4 | +5 | +6 | +7 |

> **Trend:** Highest oxidation number generally **increases across the period**.

### 3.5 Stock Notation

Proposed by German chemist **Alfred Stock**. The oxidation number is expressed as a Roman numeral in parenthesis after the metal symbol:

| Compound | Stock Name |
|----------|-----------|
| AuCl | Au(I)Cl → Gold(I) chloride |
| AuCl₃ | Au(III)Cl₃ → Gold(III) chloride |
| SnCl₂ | Sn(II)Cl₂ → Tin(II) chloride |
| SnCl₄ | Sn(IV)Cl₄ → Tin(IV) chloride |
| FeCl₂ | Fe(II)Cl₂ |
| FeCl₃ | Fe(III)Cl₃ |
| Hg₂Cl₂ | Hg₂(I)Cl₂ → reduced form of Hg(II)Cl₂ |

### 3.6 Redox Definitions Using Oxidation Number

> **Oxidation:** An **increase** in oxidation number of an element in a given substance.
>
> **Reduction:** A **decrease** in oxidation number of an element in a given substance.
>
> **Oxidising agent:** Reagent that **increases** the oxidation number of another element (i.e., gains electrons; its own O.N. **decreases**).
>
> **Reducing agent:** Reagent that **decreases** the oxidation number of another element (i.e., loses electrons; its own O.N. **increases**).
>
> **Redox reactions:** Reactions that involve **change** in oxidation number of interacting species.

### 3.7 The Paradox of Fractional Oxidation Numbers

Sometimes, calculated oxidation numbers are fractions:

| Species | Avg. O.N. | Reality |
|---------|-----------|---------|
| C₃O₂ | C = +4/3 | Two C at +2, one C at 0 |
| Br₃O₈ | Br = +16/3 | Two Br at +6, one Br at +4 |
| Na₂S₄O₆ | S = +2.5 | Two S at +5, two S at 0 |

> **Key Point:** Fractional oxidation number is always the **average**. In reality, each atom has a whole-number oxidation state. The real picture is revealed only by the structure of the molecule.
>
> Other examples: Fe₃O₄ (Fe = +8/3 → really Fe²⁺ and Fe³⁺), Mn₃O₄, Pb₃O₄.
>
> Exceptions that ARE genuinely fractional: O₂⁺ (O = +½) and O₂⁻ (O = –½).

## SECTION 4 — TYPES OF REDOX REACTIONS `[Board · NEET · JEE]`

### 4.1 Combination (Synthesis) Reactions

**General form: A + B → C**

At least one of A or B must be in elemental form. All **combustion reactions** using elemental O₂ are redox reactions.

```
  0   0       +4 –2
C(s) + O₂(g) → CO₂(g)               (7.24)

   0    0      +2 –3
3Mg(s) + N₂(g) → Mg₃N₂(s)           (7.25)

  –4+1    0     +4 –2  +1 –2
CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)
```

> ⚠️ Note: Not all combination reactions are redox (e.g., CaO + CO₂ → CaCO₃ — no change in O.N.)

### 4.2 Decomposition Reactions

**Opposite of combination.** A compound breaks into two or more components, **at least one of which must be elemental**.

```
  +1 –2           0    0
2H₂O(l) → 2H₂(g) + O₂(g)           (7.26)

  +1 –1        0    0
2NaH(s) → 2Na(s) + H₂(g)            (7.27)

 +1 +5 –2        +1 –1    0
2KClO₃(s) → 2KCl(s) + 3O₂(g)        (7.28)
```

> ⚠️ Not all decomposition reactions are redox. Example: CaCO₃(s) → CaO(s) + CO₂(g) — K, Ca, C all retain their oxidation states; **not a redox reaction**.

### 4.3 Displacement Reactions

**General form: X + YZ → XZ + Y**

An ion/atom in a compound is replaced by an ion/atom of another element.

#### (a) Metal Displacement

A more active metal displaces a less active metal from its salt solution:

```
 +2+6–2    0       0    +2+6–2
CuSO₄(aq) + Zn(s) → Cu(s) + ZnSO₄(aq)   (7.29)

 +5–2     0      0   +2–2
V₂O₅(s) + 5Ca(s) → 2V(s) + 5CaO(s)       (7.30)

 +4–1     0    0   +2–1
TiCl₄(l) + 2Mg(s) → Ti(s) + 2MgCl₂(s)   (7.31)

 +3–2     0    +3–2    0
Cr₂O₃(s) + 2Al(s) → Al₂O₃(s) + 2Cr(s)   (7.32)
```

> **Application:** Metallurgical processes — extracting pure metals from their ores.

#### (b) Non-Metal Displacement — Hydrogen Displacement

Very active metals (alkali metals, Ca, Sr, Ba) displace H from cold water:

```
2Na(s) + 2H₂O(l) → 2NaOH(aq) + H₂(g)    (7.33)
Ca(s) + 2H₂O(l) → Ca(OH)₂(aq) + H₂(g)   (7.34)
```

Less active metals (Mg, Fe) react with **steam**:

```
Mg(s) + 2H₂O(l) → Mg(OH)₂(s) + H₂(g)    (7.35)
2Fe(s) + 3H₂O(l) → Fe₂O₃(s) + 3H₂(g)    (7.36)
```

Many metals displace H from **acids**:

```
Zn(s) + 2HCl(aq) → ZnCl₂(aq) + H₂(g)    (7.37)
Mg(s) + 2HCl(aq) → MgCl₂(aq) + H₂(g)    (7.38)
Fe(s) + 2HCl(aq) → FeCl₂(aq) + H₂(g)    (7.39)
```

> Rate of H₂ evolution reflects metal reactivity: Mg > Fe (fastest to slowest)
> Ag, Au — too unreactive, do not react even with HCl.

#### (c) Non-Metal Displacement — Halogen Displacement

Halogen reactivity (as oxidising agents) decreases: **F₂ > Cl₂ > Br₂ > I₂**

```
Cl₂(g) + 2KBr(aq) → 2KCl(aq) + Br₂(l)   (7.41)
Cl₂(g) + 2KI(aq) → 2KCl(aq) + I₂(s)     (7.42)
Br₂(l) + 2I⁻(aq) → 2Br⁻(aq) + I₂(s)    (7.43)
```

> F₂ is SO reactive it attacks water: 2H₂O(l) + 2F₂(g) → 4HF(aq) + O₂(g) — so F₂ displacement reactions are NOT done in aqueous solution.
>
> **Layer Test:** Br₂ and I₂ dissolve in CCl₄ giving coloured layers → used to identify Br⁻ and I⁻ in lab.
>
> **F⁻ → F₂:** Cannot be done chemically. Only by **electrolysis** (F₂ is strongest oxidising agent; no chemical can oxidise F⁻).

### 4.4 Disproportionation Reactions

**A special type of redox reaction** in which **the same element is simultaneously oxidised and reduced.**

Requirements:
* The element must exist in **at least 3 oxidation states**
* The reacting element is in the **intermediate oxidation state**; it forms both a higher and lower oxidation state

**Classic examples:**

```
+1 –1             +1 –2    0
2H₂O₂(aq) → 2H₂O(l) + O₂(g)         (7.45)
[O in H₂O₂ is –1; goes to –2 in H₂O (reduction) and 0 in O₂ (oxidation)]

  0                      –3         +1
P₄(s) + 3OH⁻(aq) + 3H₂O(l) → PH₃(g) + 3H₂PO₂⁻(aq)  (7.46)

  0                  –2        +2
S₈(s) + 12OH⁻(aq) → 4S²⁻(aq) + 2S₂O₃²⁻(aq) + 6H₂O(l)  (7.47)

  0                +1         –1
Cl₂(g) + 2OH⁻(aq) → ClO⁻(aq) + Cl⁻(aq) + H₂O(l)    (7.48)
```

> Reaction (7.48) explains **household bleaching**: ClO⁻ (hypochlorite) oxidises colour-bearing stains to colourless compounds.

**Fluorine exception:**

F₂ + 2OH⁻ → 2F⁻ + OF₂ + H₂O (not disproportionation — F₂ is reduced to F⁻, oxygen is oxidised)

> **Why?** F is the most electronegative element — it **cannot exhibit a positive oxidation state**, so it cannot be oxidised. Hence fluorine **does NOT disproportionate**.

**Disproportionation of chlorine oxoanions:**

| Species | Cl O.N. | Can Disproportionate? | Reaction |
|---------|---------|----------------------|---------|
| ClO⁻ | +1 | Yes (can go to –1 and +5) | 3ClO⁻ → 2Cl⁻ + ClO₃⁻ |
| ClO₂⁻ | +3 | Yes | 6ClO₂⁻ → 4ClO₃⁻ + 2Cl⁻ |
| ClO₃⁻ | +5 | Yes | 4ClO₃⁻ → Cl⁻ + 3ClO₄⁻ |
| ClO₄⁻ | +7 | **No** — already at highest O.N., cannot be oxidised further |  |

## SECTION 5 — BALANCING REDOX REACTIONS `[Board · NEET · JEE]`

Two methods are used:
1. **Oxidation Number Method**
2. **Half-Reaction (Ion-Electron) Method**

### 5.1 Oxidation Number Method — Steps

**Step 1:** Write correct formulas for reactants and products.

**Step 2:** Assign oxidation numbers to all elements; identify which change.

**Step 3:** Calculate increase/decrease in O.N. per atom. Multiply by appropriate numbers to make total increase = total decrease.

**Step 4:** In aqueous medium, add H⁺ (acidic) or OH⁻ (basic) to balance ionic charges.

**Step 5:** Add H₂O molecules to balance H atoms; check O atoms.

**Worked Example — K₂Cr₂O₇ + Na₂SO₃ in acid:**

```
Step 1: Cr₂O₇²⁻(aq) + SO₃²⁻(aq) → Cr³⁺(aq) + SO₄²⁻(aq)

Step 2:
  +6        +4          +3       +6
Cr₂O₇²⁻ + SO₃²⁻ → Cr³⁺ + SO₄²⁻

Step 3:
  Cr: +6 → +3 (decrease of 3, × 2 Cr atoms = total decrease 6)
  S: +4 → +6 (increase of 2, × 3 S atoms = total increase 6) ✓
  → Cr₂O₇²⁻ + 3SO₃²⁻ → 2Cr³⁺ + 3SO₄²⁻

Step 4: Acidic medium; charges: L = –2–6 = –8; R = +6–6 = 0
  Add 8H⁺ to L: Cr₂O₇²⁻ + 3SO₃²⁻ + 8H⁺ → 2Cr³⁺ + 3SO₄²⁻

Step 5: Count H → 8H on L; add 4H₂O on R:
  Cr₂O₇²⁻(aq) + 3SO₃²⁻(aq) + 8H⁺(aq) → 2Cr³⁺(aq) + 3SO₄²⁻(aq) + 4H₂O(l) ✓
```

### 5.2 Half-Reaction (Ion-Electron) Method — Steps

**Step 1:** Write unbalanced ionic equation.

**Step 2:** Separate into oxidation and reduction half-reactions.

**Step 3:** Balance atoms other than O and H in each half-reaction.

**Step 4:**
* **Acidic medium:** Add H₂O to balance O; add H⁺ to balance H.
* **Basic medium:** Same as acidic, then add equal OH⁻ to both sides for each H⁺; combine H⁺ + OH⁻ → H₂O.

**Step 5:** Add electrons to balance charges.

**Step 6:** Multiply half-reactions to equalise electrons; add together and cancel electrons.

**Step 7:** Verify atoms and charges on both sides.

**Worked Example — Fe²⁺ + Cr₂O₇²⁻ in acid:**

```
Step 1: Fe²⁺(aq) + Cr₂O₇²⁻(aq) → Fe³⁺(aq) + Cr³⁺(aq)

Step 2:
  Oxidation: Fe²⁺ → Fe³⁺
  Reduction: Cr₂O₇²⁻ → Cr³⁺

Step 3: Balance Cr: Cr₂O₇²⁻ → 2Cr³⁺

Step 4: Balance O with H₂O, H with H⁺:
  Cr₂O₇²⁻ + 14H⁺ → 2Cr³⁺ + 7H₂O

Step 5: Balance charge:
  Fe²⁺ → Fe³⁺ + e⁻                               [× 6]
  Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O          [× 1]

Step 6: Add:
  6Fe²⁺ + Cr₂O₇²⁻ + 14H⁺ → 6Fe³⁺ + 2Cr³⁺ + 7H₂O ✓
```

**Worked Example — MnO₄⁻ + I⁻ in basic medium:**

```
Skeletal: MnO₄⁻(aq) + I⁻(aq) → MnO₂(s) + I₂(s)

Oxidation: 2I⁻ → I₂ + 2e⁻                   (×3)
Reduction: MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂ + 4OH⁻  (×2)

Equalise electrons (6e⁻ each):
  6I⁻ → 3I₂ + 6e⁻
  2MnO₄⁻ + 4H₂O + 6e⁻ → 2MnO₂ + 8OH⁻

Final: 6I⁻(aq) + 2MnO₄⁻(aq) + 4H₂O(l) → 3I₂(s) + 2MnO₂(s) + 8OH⁻(aq) ✓
```

### 5.3 Acidic vs Basic Medium — Key Rules

| Situation | Add to balance O | Add to balance H | Charge balance |
|-----------|-----------------|-----------------|---------------|
| **Acidic** | Add H₂O | Add H⁺ | Use H⁺ |
| **Basic** | Add H₂O | Add H⁺, then add equal OH⁻ to both sides | Use OH⁻ |

> **Shortcut for basic medium:** After balancing as if acidic, for each H⁺, add one OH⁻ to both sides. Where H⁺ + OH⁻ appear on same side, combine to H₂O.

## SECTION 6 — REDOX TITRATIONS `[Board · NEET]`

Titration method used in redox systems to determine the strength of a reductant/oxidant using a **redox-sensitive indicator**.

### 6.1 Types of Indicators

**(i) Self-indicator (permanganate titrations):**
MnO₄⁻ is intensely purple. When all reductant (Fe²⁺ or C₂O₄²⁻) is consumed, the first lasting pink/purple colour appears at [MnO₄⁻] as low as 10⁻⁶ mol L⁻¹. Very sensitive endpoint.

**(ii) External indicator (dichromate titrations):**
Cr₂O₇²⁻ is not self-indicating. Uses **diphenylamine** as indicator, which is oxidised just after the equivalence point to produce an intense blue colour → signals endpoint.

**(iii) Iodometric titration (indirect):**
Used for Cu(II) estimation:

```
2Cu²⁺(aq) + 4I⁻(aq) → Cu₂I₂(s) + I₂(aq)     (7.59)
I₂(aq) + 2S₂O₃²⁻(aq) → 2I⁻(aq) + S₄O₆²⁻(aq)  (7.60)
```

Starch indicator gives intense **blue with I₂**; colour disappears when I₂ is consumed by thiosulphate → sharp endpoint.

## SECTION 7 — REDOX REACTIONS AND ELECTRODE PROCESSES `[Board · NEET · JEE]`

### 7.1 From Direct to Indirect Electron Transfer — The Daniell Cell

**Direct transfer (Zn in CuSO₄):** Zn → Zn²⁺ + 2e⁻; Cu²⁺ + 2e⁻ → Cu. Heat evolved, no useful work.

**Indirect transfer (Daniell Cell):** Electrons transferred through an external wire → **electrical energy** is produced.

**Setup:**
* Beaker 1: CuSO₄ solution + Cu rod (cathode, +)
* Beaker 2: ZnSO₄ solution + Zn rod (anode, –)
* Salt bridge: KCl/NH₄NO₃ in agar — provides electric contact without mixing solutions
* External wire with ammeter and switch

**On closing switch:**
1. Electrons flow from Zn (anode) → external wire → Cu (cathode)
2. Ions migrate through salt bridge to complete circuit
3. At anode: Zn → Zn²⁺ + 2e⁻ (oxidation)
4. At cathode: Cu²⁺ + 2e⁻ → Cu (reduction)

> **Current flows opposite to electron flow** (conventional current: cathode → anode externally).

### 7.2 Key Concepts

**Redox couple:** The oxidised and reduced forms of a substance together in a half-reaction. Written as:
> Oxidised form / Reduced form (separated by slash)
> Example: Zn²⁺/Zn ; Cu²⁺/Cu

**Electrode potential:** Potential associated with each electrode. Measure of tendency of active species to remain in oxidised or reduced form.

**Standard Electrode Potential (E°):**
* Concentration of each species = 1 M (unity)
* Gas pressure = 1 atm
* Temperature = 298 K
* Reference: **Standard hydrogen electrode (SHE) = 0.00 V**

**Sign conventions:**
* **Negative E°:** Redox couple is a **stronger reducing agent** than H⁺/H₂
* **Positive E°:** Redox couple is a **weaker reducing agent** than H⁺/H₂ (equivalently, a good oxidising agent)

### 7.4 Limitations of Oxidation Number Concept

The concept of oxidation state/number has been evolving. In recent understanding:
* **Oxidation** = **decrease in electron density** around the atom(s)
* **Reduction** = **increase in electron density** around the atom(s)

## SECTION 8 — SOLVED NCERT EXAMPLES (ANNOTATED) `[Board · NEET]`

### Problem 7.1 — Identifying Oxidised/Reduced Species

**Reactions:**
(i) H₂S(g) + Cl₂(g) → 2HCl(g) + S(s)
(ii) 3Fe₃O₄(s) + 8Al(s) → 9Fe(s) + 4Al₂O₃(s)
(iii) 2Na(s) + H₂(g) → 2NaH(s)

**Solutions:**
(i) H₂S oxidised (electronegative Cl added to H, removing H from S). Cl₂ reduced (H added to Cl).
(ii) Al oxidised (O added to Al). Fe₃O₄ reduced (O removed from Fe).
(iii) Na is oxidised (forms Na⁺ in ionic NaH). H₂ is reduced (H acquires –1 oxidation state as H⁻).

### Problem 7.4 — Redox in 2Cu₂O + Cu₂S → 6Cu + SO₂

```
     +1 –2      +1 –2       0      +4 –2
2Cu₂O(s) + Cu₂S(s) → 6Cu(s) + SO₂(g)
```

* Cu: +1 → 0 (reduced), Cu₂O is **oxidant**
* S: –2 → +4 (oxidised), Cu₂S is **reductant**

### Problem 7.5 — Disproportionation of ClO⁻, ClO₂⁻, ClO₃⁻

* ClO₄⁻ does NOT disproportionate (Cl at +7 = highest O.N., cannot be further oxidised)
* ClO⁻ (+1): 3ClO⁻ → 2Cl⁻ + ClO₃⁻ (Cl goes to –1 and +5)
* ClO₂⁻ (+3): 6ClO₂⁻ → 4ClO₃⁻ + 2Cl⁻ (Cl goes to +5 and –1)
* ClO₃⁻ (+5): 4ClO₃⁻ → Cl⁻ + 3ClO₄⁻ (Cl goes to –1 and +7)

## SECTION 9 — KEY DISTINCTIONS `[NEET · JEE]`

| Feature | Oxidation | Reduction |
|---------|-----------|-----------|
| Classical (O) | Addition of O | Removal of O |
| Classical (H) | Removal of H | Addition of H |
| Classical (electronegativity) | Addition of electroneg. element | Removal of electroneg. element |
| Electronic | Loss of electrons | Gain of electrons |
| O.N. method | Increase in O.N. | Decrease in O.N. |
| Electron density (modern) | Decrease in e⁻ density | Increase in e⁻ density |

| Feature | Oxidising agent | Reducing agent |
|---------|-----------------|----------------|
| What it does | Accepts electrons; causes oxidation of other | Donates electrons; causes reduction of other |
| To itself | Gets **reduced** | Gets **oxidised** |
| O.N. change | O.N. **decreases** | O.N. **increases** |
| In electrode cell | At cathode (gains e⁻) | At anode (loses e⁻) |

| Reaction type | Key identifier |
|--------------|---------------|
| Combination | A + B → C; one/both elemental |
| Decomposition | C → A + B; one product elemental |
| Displacement | One element replaces another in compound |
| Disproportionation | Same element simultaneously oxidised AND reduced |
