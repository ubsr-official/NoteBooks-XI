## 📊 Key Numbers and Data to Memorise

```
GENERAL FORMULAS:
  Alkanes   → CₙH₂ₙ₊₂
  Alkenes   → CₙH₂ₙ    (one C=C)
  Alkynes   → CₙH₂ₙ₋₂  (one C≡C)
  Benzene   → CₙH₂ₙ₋₆  (one benzene ring)
  Cycloalkanes → CₙH₂ₙ

HYBRIDISATION:
  Alkanes: sp³  → 109.5° → tetrahedral
  Alkenes: sp²  → 120°   → trigonal planar
  Alkynes: sp   → 180°   → linear

BOND LENGTHS:
  C–C  = 154 pm    C–H (alkane) = 112 pm
  C=C  = 134 pm    C–H (alkene) = 110 pm
  C≡C  = 120 pm

BOND ENTHALPIES:
  C–C  = 348 kJ mol⁻¹
  C=C  = 681 kJ mol⁻¹
  C≡C  = 823 kJ mol⁻¹
  Benzene C–C = 139 pm (between single and double)

ENERGY DIFFERENCE (ethane conformers):
  Staggered ← 12.5 kJ mol⁻¹ → Eclipsed

HÜCKEL'S RULE NUMBERS:
  n=0: 2π;  n=1: 6π (benzene);  n=2: 10π (naphthalene);  n=3: 14π (anthracene)

HALOGEN REACTIVITY (addition to alkenes):
  F₂ > Cl₂ > Br₂ > I₂  (I₂ does NOT add under normal conditions)

H–X BOND STRENGTHS (HBr peroxide effect):
  H–Cl = 430.5 kJ mol⁻¹  (too strong → no peroxide effect)
  H–Br = 363.7 kJ mol⁻¹  (cleaved by radicals → peroxide effect ✓)
  H–I  = 296.8 kJ mol⁻¹  (too weak → I• combine to give I₂)

ALKANE PHYSICAL STATE (at 298 K):
  C₁–C₄  → gases
  C₅–C₁₇ → liquids
  C₁₈+   → solids

ALKYNE PHYSICAL STATE:
  C₂–C₄  → gases
  C₅–C₁₂ → liquids
  C₁₃+   → solids

COMBUSTION ENTHALPIES:
  CH₄ combustion: ΔHc = –890 kJ mol⁻¹
  C₄H₁₀ combustion: ΔHc = –2875.84 kJ mol⁻¹
```

```
              MONOSUBSTITUTED BENZENE + E⁺
                           |
              ┌────────────┴────────────┐
        o/p DIRECTORS              m DIRECTORS
              |                         |

              |                         |

  Electron-donating groups → o/p → activating
  Electron-withdrawing → m → deactivating
  EXCEPTION: Halogens → o/p directors BUT deactivating
    (–I effect > +R effect overall → net deactivating)
    (BUT +R effect makes o/p relatively richer than m)
```

# 📝 ONE-PAGE SUPER CONDENSED NOTES

## Alkanes
**Formula:** CₙH₂ₙ₊₂ | **Hybridisation:** sp³ | **Bond angle:** 109.5°
**Key reaction:** Free radical halogenation (hν or Δ) — 3°H replaced fastest
**Mechanism:** Initiation (Cl–Cl homolysis) → Propagation (Cl• + CH₄ → •CH₃ + HCl; •CH₃ + Cl₂ → CH₃Cl + Cl•) → Termination
**Conformations:** Staggered (stable, min. torsional strain) > Eclipsed (unstable); energy gap = 12.5 kJ mol⁻¹; cannot be isolated

## Alkenes
**Formula:** CₙH₂ₙ | **Hybridisation:** sp² | **Bond angle:** 120°
**Key reaction:** Electrophilic addition to π bond
**Markovnikov:** H⁺ → more substituted C (via stable 2°/3° carbocation)
**Anti-Markov (peroxide):** Br• → terminal C (via stable 2° radical) | **ONLY HBr!**
**Tests:** Br₂/CCl₄ decolourises; KMnO₄ decolourises (Baeyer's test)
**cis-trans:** Requires different groups on each sp² C; cis has higher µ; trans has higher MP

## Alkynes
**Formula:** CₙH₂ₙ₋₂ | **Hybridisation:** sp | **Bond angle:** 180°
**Acidity:** Terminal H (sp C–H) → acidic; reacts with Na, NaNH₂; Ethyne > ethene > ethane
**Hydration:** + H₂O/Hg²⁺/H⁺ → carbonyl compound (ethyne → ethanal; propyne → propanone)
**Cyclic polymerisation:** 3HC≡CH → benzene (Fe, 873K)

## Benzene & Aromaticity
**Formula:** C₆H₆ | **Hybridisation:** sp² | **Bond:** 139 pm (all equal)
**Aromaticity:** Planar + fully delocalised + (4n+2)π electrons
**Key reactions:** SE reactions (nitration, halogenation, sulphonation, F-C alkylation/acylation)
**Mechanism:** E⁺ generated → σ-complex (arenium ion) → H⁺ lost → product
**Directive effects:** o/p directors (e-donating: –OH, –NH₂, alkyl) | m-directors (e-withdrawing: –NO₂, –CN, –COOH) | Halogens: o/p but deactivating

## ⚠️ COMMON EXAM MISTAKES / TRAPS

| Mistake | Correction |
|---------|-----------|
| Saying Wurtz reaction gives odd-carbon alkanes easily | Wurtz with **one type** of RX gives even-carbon alkanes; two different RX gives **mixture of 3 alkanes** |
| Forgetting peroxide effect is ONLY HBr | HCl (too strong C–H, no radical) and HI (I• recombine to I₂) do NOT show peroxide effect |
| Saying all alkynes are acidic | Only **terminal alkynes** (H on sp carbon) are acidic; internal alkynes (R–C≡C–R) are not |
| Confusing cis/trans stability | cis has higher **dipole moment and BP**; trans has higher **melting point** (better packing) |
| Saying halogen is activating in SE | Halogens are **o/p directors** but **deactivating** (slow the reaction but direct to o/p) |
| Forgetting IUPAC: anti-Markovnikov product | 1-Bromopropane (CH₃CH₂CH₂Br) is IUPAC 1-bromopropane; 2-bromopropane is the Markovnikov product |
| Kolbe's method for methane | Methane **cannot** be made by Kolbe's — acetate gives ethane (CH₃•+CH₃• → C₂H₆) |
| Benzene reacting like alkene | Benzene undergoes **substitution** (not addition) under normal conditions due to delocalisation |
| Lindlar's catalyst gives trans-alkene | Lindlar's → **cis**; Na/liq.NH₃ → **trans** |
| Thinking all aromatic compounds have benzene ring | Non-benzenoids (cyclopentadienyl anion, cycloheptatrienyl cation) are also aromatic |
