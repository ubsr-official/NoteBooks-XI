## 🔢 Quantum Numbers — At a Glance

| QN        | Symbol | Determines          | Values             | Range          |
| --------- | ------ | ------------------- | ------------------ | -------------- |
| Principal | n      | Shell, size, energy | 1, 2, 3, 4...      | n ≥ 1         |
| Azimuthal | l      | Subshell, shape     | 0, 1, 2 ... (n−1) | 0 ≤ l ≤ n−1 |
| Magnetic  | mₗ    | Orbital orientation | −l to +l          | (2l+1) values  |
| Spin      | ms     | Spin direction      | +½ or −½        | 2 values only  |

**l = 0 → s (sphere) | l = 1 → p (dumbbell) | l = 2 → d (cloverleaf) | l = 3 → f (complex)**

## 🗂️ Orbital Filling Order + Node Rules

```
Filling order: 1s 2s 2p 3s 3p 4s 3d 4p 5s 4d 5p 6s 4f 5d 6p 7s...
(n+l) rule: lower (n+l) fills first; equal (n+l) → lower n fills first

Radial nodes  = n − l − 1       Angular nodes = l       Total nodes = n − 1
Max e⁻/orbital = 2              Max e⁻/shell = 2n²      Max e⁻/subshell = 2(2l+1)
```

**Exceptions:** Cr → [Ar]**3d⁵4s¹** (not 3d⁴4s²) | Cu → [Ar]**3d¹⁰4s¹** (not 3d⁹4s²)

**Reason:** Extra stability of half-filled (d⁵) and fully-filled (d¹⁰) configurations.

─────────────────────────────────────────────────────────────────────────
  de BROGLIE + HEISENBERG — Quick Summary
─────────────────────────────────────────────────────────────────────────

  de Broglie (1924):            λ = h/mv = h/p
  • Every moving particle has a wave
  • Significant ONLY for microscopic particles (electrons, protons)
  • Electron: λ ≈ 10⁻¹⁰ m (measurable, X-ray scale) ✓
  • Cricket ball: λ ≈ 10⁻³⁴ m (immeasurably tiny) ✗

  Heisenberg (1927):            Δx · Δpₓ ≥ h/4π
  • Cannot simultaneously know exact position AND velocity
  • Not instrument error — FUNDAMENTAL property of nature
  • Macroscopic objects: uncertainty negligible (10⁻²⁸ m²s⁻¹)
  • Electron: uncertainty enormous (10⁻⁴ m²s⁻¹) — very real
  • This is WHY Bohr's defined orbits are physically impossible

  WHY BOHR FAILS (Heisenberg reason):
  Bohr's orbit requires BOTH exact position AND exact velocity
  of electron at every instant → violates Heisenberg Uncertainty Principle
