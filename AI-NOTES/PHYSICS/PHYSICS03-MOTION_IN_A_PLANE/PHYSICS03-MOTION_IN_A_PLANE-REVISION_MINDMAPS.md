## 🔢 Key Definitions — Absolute Must-Memorise

| Quantity | Definition | Formula | SI Unit |
|---|---|---|---|
| **Vector** | Magnitude + direction; obeys triangle law | — | depends |
| **Scalar** | Magnitude only | — | depends |
| **Unit vector** | Magnitude = 1; specifies direction | n̂ = **A**/|**A**| | dimensionless |
| **Position vector** | Vector from origin to point P | **r** = xî + yĵ | m |
| **Displacement** | Change in position (vector) | Δ**r** = **r′** − **r** | m |
| **Resultant** | Vector sum of two or more vectors | R = √(A²+B²+2ABcosθ) | — |
| **Instantaneous velocity** | dr/dt | **v** = vₓî + vᵧĵ | m s⁻¹ |
| **Centripetal acceleration** | Directed toward centre in UCM | aₒ = v²/R = ω²R | m s⁻² |
| **Angular speed** | Rate of change of angular displacement | ω = Δθ/Δt = v/R | rad s⁻¹ |
| **Time period** | Time for one complete revolution | T = 2π/ω = 1/ν | s |

## 📐 Essential Formulae — Must Know Cold

```
VECTOR ADDITION:
    ┌──────────────────────────────────────────────────────┐
    │  R = √(A² + B² + 2AB cosθ)   (magnitude)           │
    │  tan α = B sinθ/(A + B cosθ)  (direction)           │
    │                                                       │
    │  Special: θ = 0°  → R = A + B  (max)                │
    │           θ = 90° → R = √(A² + B²)                  │
    │           θ = 180°→ R = |A − B| (min)               │
    └──────────────────────────────────────────────────────┘

COMPONENTS:
    ┌──────────────────────────────────────────────────────┐
    │  Aₓ = A cosθ,    Aᵧ = A sinθ                        │
    │  A = √(Aₓ² + Aᵧ²),  tanθ = Aᵧ/Aₓ                   │
    │  R = A + B  →  Rₓ = Aₓ+Bₓ,  Rᵧ = Aᵧ+Bᵧ             │
    └──────────────────────────────────────────────────────┘

PROJECTILE MOTION (origin at launch, θ₀ = projection angle):
    ┌──────────────────────────────────────────────────────┐
    │  x = (v₀cosθ₀)t         [horizontal: const. speed] │
    │  y = (v₀sinθ₀)t − ½gt²  [vertical: free fall]      │
    │  vₓ = v₀cosθ₀ = const.                              │
    │  vᵧ = v₀sinθ₀ − gt                                  │
    │                                                       │
    │  hₘ  = (v₀sinθ₀)²/(2g)   [max height]              │
    │  Tƒ  = 2v₀sinθ₀/g        [time of flight]          │
    │  R   = v₀²sin2θ₀/g       [range]                   │
    │  Rₘₐₓ = v₀²/g            [max range at θ=45°]      │
    └──────────────────────────────────────────────────────┘

UNIFORM CIRCULAR MOTION:
    ┌──────────────────────────────────────────────────────┐
    │  v = Rω = 2πRν = 2πR/T                              │
    │  aₒ = v²/R = ω²R = 4π²ν²R                          │
    │  ω = 2πν = 2π/T                                      │
    │  Direction of aₒ: toward CENTRE always              │
    └──────────────────────────────────────────────────────┘
```

## 📊 Important Comparisons — Instant Recall

```
PROJECTILE vs CIRCULAR:
───────────────────────────────────────────────────────
Projectile:   Acceleration constant (g, downward)
              Path: parabola
              v and a NOT perpendicular generally

Circular:     Speed constant; velocity direction changes
              Acceleration toward centre (centripetal)
              v ⊥ aₒ  ALWAYS (90° between them)
              Kinematic equations DO NOT apply

AVERAGE SPEED vs |AVERAGE VELOCITY|:
  average speed ≥ |average velocity|
  (same as in Ch. 2 — still holds in 2D!)

PATH vs DISPLACEMENT (in 2D too):
  |displacement| ≤ path length
```

## ⚠️ Critical Distinctions — High-Yield Traps

```
PROJECTILE MOTION TRAPS:
  ✓ At max height: vᵧ = 0, but vₓ = v₀cosθ₀ ≠ 0
  ✓ At max height: speed = v₀cosθ₀ (NOT zero)
  ✓ At max height: a = g downward (NOT zero)
  ✓ Horizontal acceleration = 0 throughout
  ✓ Trajectory = parabola (confirm by eliminating t)
  ✓ (45°+α) and (45°−α) → same range
  ✓ θ and (90°−θ) → same range, different heights

CIRCULAR MOTION TRAPS:
  ✓ |aₒ| = constant; direction of aₒ = NOT constant
  ✓ aₒ is NOT a constant vector
  ✓ Speed = constant; velocity = NOT constant
  ✓ Kinematic equations INVALID for UCM
  ✓ v ⊥ r only for uniform circular (not general curved)

VECTOR ALGEBRA TRAPS:
  ✓ |A + B| ≠ A + B generally (equality only when θ = 0°)
  ✓ |A − B| ≠ |A| − |B| generally
  ✓ A + B = A + B only if same direction
  ✓ A·(-A) = 0  (null vector)
  ✓ Equal magnitudes ≠ equal vectors (direction must match)
```
