## 🔢 Key Definitions — Absolute Must-Memorise

| Quantity | Definition | Formula | SI Unit |
|---|---|---|---|
| **Inertia** | Resistance to change in state of rest/motion | — | measured by mass |
| **Momentum** | Product of mass and velocity | p = mv | kg m s⁻¹ = N s |
| **Force** | Cause that changes or tends to change state of motion | F = dp/dt = ma | N = kg m s⁻² |
| **Impulse** | Large force × short time = change in momentum | J = FΔt = Δp | N s |
| **Normal Force** | Contact force ⊥ to surfaces in contact | N = mg (equilibrium, horiz.) | N |
| **Static Friction** | Friction opposing impending relative motion | fs ≤ μsN | N |
| **Kinetic Friction** | Friction opposing actual relative motion | fk = μkN | N |
| **Centripetal Force** | Net inward force in circular motion | fc = mv²/R | N |
| **Optimum Speed** | Speed on banked road needing zero friction | v₀ = √(Rg tanθ) | m s⁻¹ |

## 📐 Essential Formulae — Must Know Cold

```
NEWTON'S LAWS:
    ┌──────────────────────────────────────────────────────┐
    │  1st Law: ΣF = 0 ↔ a = 0   (defines inertia/force) │
    │  2nd Law: F = dp/dt = ma    (1 N = 1 kg m s⁻²)     │
    │  3rd Law: F_AB = −F_BA      (pairs; diff. bodies)   │
    └──────────────────────────────────────────────────────┘

MOMENTUM:
    ┌──────────────────────────────────────────────────────┐
    │  p = mv                [MLT⁻¹]                      │
    │  Impulse = FΔt = Δp    [MLT⁻¹]                     │
    │  Conservation: Σp = const  when ΣF_ext = 0          │
    │  Gun recoil: m_b·v_b + m_g·v_g = 0                  │
    └──────────────────────────────────────────────────────┘

FRICTION:
    ┌──────────────────────────────────────────────────────┐
    │  fs ≤ μsN  (static; self-adjusting up to μsN)       │
    │  fk = μkN  (kinetic; constant)                      │
    │  μk < μs   ALWAYS                                   │
    │  Angle of repose: μs = tanθr                        │
    │  On incline: slides when θ > θr = tan⁻¹(μs)        │
    └──────────────────────────────────────────────────────┘

CIRCULAR MOTION:
    ┌──────────────────────────────────────────────────────┐
    │  Centripetal force: fc = mv²/R                      │
    │  Level road vmax: √(μsRg)    [indep. of mass]       │
    │  Banked v₀ (f=0): √(Rg tanθ)                       │
    │  Banked vmax: √[Rg(tanθ+μs)/(1−μs tanθ)]           │
    │  Banked vmin: √[Rg(tanθ−μs)/(1+μs tanθ)]           │
    └──────────────────────────────────────────────────────┘

LIFT (ELEVATOR):
    ┌──────────────────────────────────────────────────────┐
    │  Rest/uniform velocity:   N = mg                    │
    │  Accelerating up (+a):    N = m(g+a)  [heavier]    │
    │  Accelerating down (+a):  N = m(g−a)  [lighter]    │
    │  Free fall:               N = 0        [weightless] │
    └──────────────────────────────────────────────────────┘
```

## 📊 Important Comparisons — Instant Recall

```
STATIC vs KINETIC FRICTION:
──────────────────────────────────────────────────────────
Static:   Opposes IMPENDING motion; self-adjusting (0→μsN)
Kinetic:  Opposes ACTUAL motion; constant = μkN
          μk < μs  ALWAYS
          Both independent of area of contact
          Both approximately independent of normal force (law)

1st LAW vs 2nd LAW:
  1st Law: Defines inertia + inertial frames; special case F=0
  2nd Law: Quantifies force-acceleration relation; assumes
           inertial frame already established by 1st law
  They are NOT equivalent — 1st law has extra conceptual content

ACTION-REACTION (3rd Law) vs EQUILIBRIUM (1st Law):
  Action-Reaction: Different bodies; simultaneous; never cancel
  Equilibrium:     Same body; net force = 0; may cancel
  mg and N on a book: NOT action-reaction (same body)
  N on book by table AND N on table by book: IS action-reaction
```

## ⚠️ Critical Distinctions — High-Yield Traps

```
MOMENTUM AND IMPULSE TRAPS:
  ✓ Impulse = Δp = FΔt (large F, small t)
  ✓ Momentum is a vector; change in direction → change in momentum
  ✓ Ball bouncing off wall: |Δp| = 2mv (NOT zero; direction changes)
  ✓ [Impulse] = [Momentum] = [MLT⁻¹] ≠ [Force] = [MLT⁻²]

FRICTION TRAPS:
  ✓ fs ≠ μsN in general; fs = μsN ONLY at the limiting condition
  ✓ Friction opposes RELATIVE MOTION, not absolute motion
  ✓ Static friction can accelerate a body (e.g., box in train)
  ✓ Rolling friction << kinetic friction (orders of magnitude)
  ✓ Friction laws are empirical (approximate), not fundamental

CIRCULAR MOTION TRAPS:
  ✓ Centripetal force is NOT a new force — name for net inward force
  ✓ Never add "centripetal force" as a separate entity in FBD
  ✓ vmax on level road is independent of vehicle mass
  ✓ On banked road at v₀: friction = 0 (not maximum)
  ✓ Below vmin (banked): car slides INWARD and down
  ✓ Above vmax (banked): car slides OUTWARD and up

NEWTON'S THIRD LAW TRAPS:
  ✓ Action and reaction are SIMULTANEOUS (no cause-effect order)
  ✓ They act on DIFFERENT bodies — never cancel each other
  ✓ Internal forces (A→B and B→A within system) cancel in pairs
  ✓ mg and N on a body are NOT an action-reaction pair
  ✓ Horse can move forward despite reaction from cart —
    net external force (ground friction on horse) is unbalanced
```

## 🔁 Friction on an Incline — Decision Chart

```
BLOCK ON INCLINED PLANE (angle θ, coeff μs, μk):
─────────────────────────────────────────────────────
Step 1: Find angle of repose θr = tan⁻¹(μs)

Step 2: Compare θ with θr:

  θ < θr  → Block is STATIONARY (static friction balances gravity)
             fs = mg sinθ  (less than maximum)
             N = mg cosθ

  θ = θr  → Block is on VERGE of sliding (limiting static friction)
             fs = μs N = mg sinθ = (fs)max

  θ > θr  → Block SLIDES (kinetic friction acts)
             fk = μk N = μk mg cosθ  (opposing motion, up the incline)
             Net force = mg sinθ − μk mg cosθ = mg(sinθ − μk cosθ)
             Acceleration = g(sinθ − μk cosθ)   [down the incline]
─────────────────────────────────────────────────────
Note: μk < μs → fk < (fs)max → acceleration when sliding
      is greater than zero (cannot decelerate back to rest once moving)
```

# 🗺️ MIND MAP 5 — Free Body Diagram Strategy

```
                    ┌──────────────────────────┐
                    │  FREE BODY DIAGRAM (FBD)  │
                    └────────────┬─────────────┘
                                 │
                    ┌────────────▼─────────────┐
                    │  1. ISOLATE the body      │
                    │     (your chosen system)  │
                    └────────────┬─────────────┘
                                 │
                    ┌────────────▼─────────────┐
                    │  2. SHOW ALL forces ON    │
                    │     the body:             │
                    │   • Weight mg (↓ always)  │
                    │   • Normal N (⊥ surface)  │
                    │   • Tension T (↑ or →)    │
                    │   • Friction f (opposes   │
                    │     relative motion)      │
                    │   • Applied forces        │
                    └────────────┬─────────────┘
                                 │
                    ┌────────────▼─────────────┐
                    │  3. DO NOT include forces │
                    │     the body exerts on    │
                    │     other bodies          │
                    └────────────┬─────────────┘
                                 │
                    ┌────────────▼─────────────┐
                    │  4. APPLY Newton's 2nd    │
                    │     Law along each axis:  │
                    │   ΣFx = max               │
                    │   ΣFy = may               │
                    │   (or ΣF = 0 if equil.)   │
                    └──────────────────────────┘

COMMON MISTAKES IN FBDs:
  ✗ Including reaction forces on OTHER bodies
  ✗ Adding "centripetal force" as extra force
  ✗ Using mg = N blindly on inclines or in lifts
  ✗ Forgetting friction on inclined surfaces
  ✗ Wrong direction of tension (should pull, not push)
```

## 🏆 Last-Minute Exam Checklist

```
Before answering any problem in Laws of Motion:

□ Have I drawn a free-body diagram?
□ Is the friction direction correct (opposing relative/impending motion)?
□ On an incline: is N = mg cosθ (NOT mg)?
□ In a lift: is N ≠ mg when accelerating?
□ For circular motion: have I identified the REAL force acting as centripetal?
□ Is friction in circular motion STATIC (not kinetic, unless given)?
□ Have I checked μk < μs — have I used the right coefficient?
□ For momentum conservation: is the system truly isolated (ΣFext = 0)?
□ Is the impulse = Δp = 2mv (for reversal) or mv (for coming to rest)?
□ Are action-reaction forces on DIFFERENT bodies — not same body?
```
